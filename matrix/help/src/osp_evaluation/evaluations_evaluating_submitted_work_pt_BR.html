Avaliar Trabalho Submetido

Para rever o trabalho submetido, acesse a ferramenta Avaliações e siga estes passos:

  1. Se a página inicial de Avaliações não for exibida, clique no botão Reset.
  2. Clique no título desejado na lista. O Sakai exibirá a célula da matriz apropriada ou página do assistente para o título que você selecionou.
  3. Leia as instruções e todos os formularios e outros anexos que foram submetidos. Nota: Para retornar à página inicial de Avaliações, a qualquer momento, clique no botão Reset.
  4. Na seção Avaliação, clique no link Adicionar Avaliação.
  5. Preencha as caixas no formulário. Certifique-se de introduzir um título para sua avaliação. Essa entrada é obrigatória.
  6. Para adicionar os seus comentários, clique no botão Adicionar. O Sakai anexará os seus comentários para a página do assistente ou participante da matriz de células e exibirá novamente a página inicial Avaliações. Ou, para retornar a essa página sem anexar comentários, clique em Cancelar.
