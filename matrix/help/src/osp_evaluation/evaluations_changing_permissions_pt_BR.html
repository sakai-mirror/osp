Alterar Permissões

As permissões são mais complexas para os avaliadores do que para a maioria dos outros membros do site. Para você atuar como avaliador de trabalhos em uma célula de matriz específica ou página do assistente, as seguintes condições devem ser atendidas:

  • Você precisa ter o papel de avaliador neste site.
  • Na página específica de células ou assistente da matriz, o seu papel ou seu nome de usuário deve estar especificado como um avaliador.
  • Seu papel deve ter permissão de "avaliar" nas ferramentas Matrizes ou Assistentes e Avaliações

Para alterar as permissões padrão para a ferramenta Avaliações, acesse a ferramenta e siga estes passos:

  1. Se a barra de menu de Avaliações não for exibida, clique no botão Reset.
  2. Clique no botão Permissões na barra de menu. O Sakai exibirá a tela de permissões.
  3. Referindo à tabela abaixo, marque as caixas para selecionar as permissões adequadas para os usuários em cada papel.



    Permissão Descrição
    View Owner Os usuários neste papel podem ver o nome do usuário, cujo trabalho está sendo avaliado em uma célula de matriz ou página do assistente.
    Evaluate Os usuários neste papel podem rever o trabalho concluído pelos participantes do site em matrizes e assistentes e oferecer feedback somativo sobre ele.


  4. Para salvar as configurações de permissões, clique em salvar. O Sakai salvará as configurações e exibirá a página inicial Avaliações. Ou, para voltar para a tela sem salvar as alterações, clique em cancelar.
