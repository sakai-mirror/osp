Modifications des permissions

Les permissions sont plus complexes pour les évaluateurs que pour les autres membres du site. Pour agir comme évaluateur d'un travail dans une matrice spécifique ou une page de l'assistant, les conditions suivantes sont requises :

  • Vous devez être évaluateur sur ce site.
  • Sur la cellule de la matrice ou la page de l'assistant, votre rôle ou votre identifiant d'utilisateur doit être spécifié comme évaluateur.
  • Vous devez avoir la permission "évaluer" dans l'outil des matrices ou des assistants et aussi dans l'outil d'évaluation.

Pour changer les permissions par défaut de l'outil d'évaluation, accédez à l'outil en suivant ces étapes :

  1. Si la barre du menu d'évaluation n'est pas visible, cliquez sur le bouton "Réinitialiser".
  2. Cliquez sur le bouton "permissions" de la barre du menu. Sakai affiche la page des permissions.
  3. En vous référant au tableau ci-dessous, cochez les permissions appropriées pour chaque type d'utilisateur.



    Permission Description
    Voir auteur Les utilisateurs peuvent voir le nom de l'auteur dont le travail est évalué dans une matrice ou une page de l'assistant.
    Evaluation Les utilisateurs peuvent évaluer le travail rendu par les participants du site dans des matrices et assistants et donner un avis sur ce travail.


  4. Pour sauvegarder les réglages de permissions, cliquez sur "sauvegarde". Sakai sauvegarde les réglages et affiche la page d'accueil des évaluations. Pour retourner à la page sans sauvegardez vos changement, cliquez sur "annuler".
