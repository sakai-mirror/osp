Evaluation du travail soumis

Pour évaluer un travail soumis, accédez à l'outil d'évaluations et suivez ces étapes :

  1. Si la page d'accueil des évaluations n'est pas affichée, cliquez sur le bouton "Réinitialiser".
  2. Cliquez sur le titre désiré dans la liste. Sakai affiche la cellule de matrice appropriée ou la page de l'assistant pour le titre que vous avez sélectionné.
  3. Evaluez les instructions et toutes les formes qui ont été consignées. Note : Pour revenir à la page d'accueil des évaluations, cliquez sur le bouton "Réinitialiser" quand vous le souhaitez.
  4. Dans la section d'évaluation, cliquez sur le lien "Ajout d'une évaluation".
  5. Complétez les champs de saisie. Prenez garde à saisir un titre pour votre évaluation. Cette entrée est requise.
  6. Pour ajouter vos commentaires, cliquez sur le bouton "Ajouter". Sakai attache vos commentaires à la page de l'assistant du contributeur ou à la cellule de matrice et affiche la page d'accueil des évaluations. Sinon, pour retourner à cette page sans laisser de commentaire, cliquez sur "Annuler".
