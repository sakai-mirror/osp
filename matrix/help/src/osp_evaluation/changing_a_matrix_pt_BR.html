Alterar uma Matriz

Se você ainda não publicou uma matriz, você pode alterar qualquer parte dela. O modo Visualizar permite testar a matriz e ainda alterar qualquer parte dela. Uma vez publicada, você é limitado no que você pode mudar.

Para rever uma matriz, acesse a ferramenta Matrizes e siga estes passos:

  1. Se a barra de menu de Matrizes não for exibida, clique no botão Reset.
  2. Encontre a matriz que pretende alterar e clique no link Rever abaixo do seu nome. Sakai exibirá a tela Rever Matriz.
  3. No menu de Matrizes perto do topo da tela, clique em Editar Propriedades. Sakai exibe a tela Rever Propriedades da Matriz.
  4. Para rever uma coluna ou linha, clique no link Rever abaixo do seu nome e proceda conforme as instruções em "Criar uma Matriz",.
  5. Para remover uma coluna ou linha, clique no link Remover abaixo do seu nome. Sakai exibe a janela Remover Coluna (ou Remover Linha). Para prosseguir, clique em Continuar. Sakai exclui a coluna (ou linha) e exibirá novamente a tela Rever Matriz.
  6. Para mover uma linha ou coluna para cima ou para baixo na lista, clique no link apropriado abaixo do seu nome. Sakai reordena as colunas.
  7. Para alterar outras seleções e entradas nesta tela, proceda conforme as instruções em "Criar uma Matriz", acima.
  8. Para salvar seu trabalho, clique em Salvar alterações. Ou, para sair sem adicionar uma nova matriz, clique em Cancelar. Se você clicou em Salvar alterações, Sakai salva as alterações e exibe a matriz revisada na tela Rever Matriz.
