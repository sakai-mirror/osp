Modification d'une matrice

Tant que vous n'avez pas publié une matrice, vous pouvez la modifier sans contrainte. Le mode Aperçu vous permet de tester la matrice tout gardant la possibilité de la modifier. Une fois publiée, vous êtes limités sur les modifications possibles.

Pour modifier une matrice, accédez à l'outil Matrices et suivez les étapes suivantes :

  1. Si la page d'accueil des Matrices n'est pas affichée, cliquez sur le bouton Réinitialiser.
  2. Trouvez la matrice que vous souhaitez modifier et cliquez sur le lien Modifier sous son nom. Sakai affiche alors l'écran de modification d'une matrice.
  3. Dans le menu Matrices en haut de l'écran, cliquez sur Modifier les propriétés. Sakai affiche l'écran de modification des propriétés d'une matrice.
  4. Pour modifier une colonne ou une ligne, cliquez sur le bouton Modifier sous son nom et procédez comme indiqué dans "Création d'une matrice".
  5. Pour supprimer une colonne ou une ligne, cliquez sur le lien Supprimer sous son nom. Sakai affiche la fenêtre de suppression d'une colonne (ou d'une ligne). Pour poursuivre, cliquez sur Continuer. Sakai supprime la colonne (ou la ligne) et affiche à nouveau l'écran de modification d'une matrice.
  6. Pour déplacer une ligne ou une colonne vers le haut ou vers le bas dans la liste, cliquez sur le lien approprié sous son nom. Sakai réorganise les lignes ou les colonnes.
  7. Pour modifier d'autres sélections et d'autres entrées, procédez comme indiqué dans "Création d'une matrice" ci-dessus.
  8. Pour enregistrer votre travail, cliquez sur Enregistrer les modifications. Pour quitter sans ajouter de nouvelle matrice, cliquez sur Annuler.
