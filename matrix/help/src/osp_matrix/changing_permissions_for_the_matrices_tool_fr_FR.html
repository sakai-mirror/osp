Modification des permissions

Pour modifier les permissions par défaut de l'outil Matrices, accédez à l'outil et suivez ces étapes :

  1. Si la barre de menu des matrices n'est pas affichée, cliquez sur le bouton Réinitialiser.
  2. Cliquez sur le bouton Permissions dans la barre de menu. Sakai affiche l'écran des Permissions.
  3. En utilisant le tableau suivant, sélectionnez les permissions pour les utilisateurs de chaque rôle.



    Permission Description
    Utilisation Les utilisateurs de ce rôle peuvent accéder aux matrices, les utiliser pour déposer des documents et réfléchir sur leur travail.
    Création Les utilisateurs de ce rôle peuvent créer de nouvelles matrices et importer des matrices existantes qu'ils peuvent utiliser eux, et/ou les participants du site.
    Publication Les utilisateurs de ce rôle peuvent publier des matrices utilisables par les participants du site.
    Export Les utilisateurs de ce rôle peuvent exporter une matrice sur leur machine sous forme de fichier compressé.
    Révision Les utilisateurs de ce rôle peuvent consulter le travail en cours dans les matrices (en utilisant le menu déroulant pour aller à la matrice d'un participant du site) et donner leur avis.
    Évaluation Les utilisateurs de ce rôle peuvent consulter le travail des matrices (en utilisant le menu déroulant pour aller à la matrice d'un participant du site) et l’évaluer.


  4. Pour enregistrer les paramètres de permissions, cliquez sur Enregistrer. Sinon, cliquez sur Annuler.
