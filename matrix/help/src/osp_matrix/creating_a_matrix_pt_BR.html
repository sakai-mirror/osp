Criar uma Matriz

Criar uma matriz envolve várias etapas:

  1. Criar um framework para a matriz.
  2. Configurar as células.
  3. Publicar a matriz.

Etapa 1: Criar o Framework

Para criar o framework para uma matriz, acesse a ferramenta Matrizes e siga estes passos:

  1. Se a barra de menu de Matrizes não for exibida, clique no botão Reset.
  2. Na barra de menu Matrizes perto do topo da tela, clique no botão Adicionar. O Sakai exibirá a tela Adicionar Matriz.
  3. Nas caixas apropriadas, digite um título para a matriz e uma descrição de sua finalidade. Apenas a entrada do Título é obrigatória. Atenção: Escolha o título cuidadosamente. Após a publicação de uma matriz, você não pode alterar o título.
  4. Escolha uma das seguintes opções:
    • Para aplicar o estilo Sakai padrão para esta matriz, pule esta etapa e vá direto para a etapa 6.
    • Para aplicar outro estilo a esta matriz, clique em Selecionar Estilo. O Sakai exibe a tela Gerenciar Estilos.
      • Esta tela lista os estilos disponíveis para uso em suas matrizes.
  5. Clique no link Selecionar para o estilo que deseja aplicar a essa matriz. O Sakai exibirá novamente a tela Adicionar Matriz. O nome do estilo que você escolheu é exibido na caixa Estilo. Ou, para voltar para a tela sem selecionar um estilo, clique em Voltar.
  6. Para adicionar uma coluna, clique em Adicionar Coluna. O Sakai exibirá uma tela para edição da coluna .
  7. Na caixa Nome, digite um nome para exibição no cabeçalho da coluna. Tenha em mente que os nomes mais curtos para as colunas permitirá uma melhor utilização do espaço da tela.
  8. Para selecionar uma cor de fundo para esta coluna, diferente do padrão (cinza), clique no ícone da paleta de cores para a caixa de Cor de Fundo. Quando o Sakai exibir opções de cores, clique na cor de fundo que você deseja.
  9. Para selecionar uma cor para o texto desta coluna, diferente do padrão (preto), clique no ícone da paleta de cores para a caixa de Cor da Fonte. Quando o Sakai exibir opções de cores, clique na cor do texto que você deseja.
  10. Para salvar o seu trabalho nesta coluna, clique em Atualizar. O Sakai exibirá a tela Adicionar Matriz, incluindo informações para a coluna que você acabou de criar. Ou, para retornar à tela Adicionar Matriz sem adicionar esta coluna, clique em Cancelar.
  11. Repita os passos 6 a 10 conforme necessário para criar as colunas restantes para a sua matriz. Atenção: Pense cuidadosamente sobre o número de colunas na matriz. Após a publicação de uma matriz, você não pode alterar o número de colunas que ela contém.
  12. Para alterar a terminologia para algo diferente do que a palavra "coluna", digite o termo que você preferir na caixa de rótulo da coluna.
  13. Para adicionar uma linha, clique em Adicionar linha. O Sakai exibe a tela Editar Linha.
  14. Na caixa Nome, digite um nome para exibição no cabeçalho da linha.
  15. Para selecionar uma cor de fundo para esta linha diferente do padrão (cinza), clique no ícone da paleta de cores para a caixa de Cor de Fundo. Quando o Sakai exibir opções de cores, clique na cor de fundo que você deseja.
  16. Para selecionar uma cor para o texto nesta linha diferente do padrão (preto), clique no ícone da paleta de cores para a caixa de cor da fonte. Quando o Sakai exibir opções de cores, clique na cor do texto que você deseja.
  17. Para salvar o seu trabalho nesta linha, clique em Atualizar. O Sakai exibirá a tela Adicionar Matriz, incluindo informações sobre a linha que você acabou de criar. Ou, para retornar à tela Adicionar Matriz sem acrescentar essa linha, clique em Cancelar.
  18. Repita os passos 13 - 17 como necessários para criar as linhas restantes para a matriz. Atenção: Pense cuidadosamente sobre o número de linhas na matriz. Após a publicação de uma matriz, você não pode alterar o número de linhas que ela contém.
  19. Para alterar a terminologia para algo diferente do que a palavra "Linha", digite o termo que você preferir na caixa rótulo de linha.
  20. Sob Apresentação Matrix, clique no botão apropriado para selecionar a maneira que você quer que os participantes trabalhem através das células da matriz.
  21. Em Cores do Status da Matriz, selecione cores para indicar o estado de cada célula na matriz.



    Status Significado
    Ready Esta célula está disponível para o trabalho participante site.
    Pending O participante submeteu esta célula para a avaliação.
    Completed Esta célula foi avaliada.
    Locked Esta célula não está disponível para o trabalho do participante site neste momento.


  22. Para salvar seu trabalho, clique em Salvar alterações. Ou, para sair sem adicionar uma nova matriz, clique em Cancelar.

    Se você clicou em Salvar alterações, o Sakai salvará as alterações e apresentará a nova matriz na tela Rever Matriz.

    Observe o seguinte:
    • O título matriz que você especificou na etapa 3 é exibido no canto superior esquerdo da tabela. Se você entrar com uma descrição, será exibido acima da tabela.
    • Os nomes das colunas, cores de fundo e cores de texto inseridos nos passos 6 - 11 são exibidos no cabeçalho das colunas.
    • Os nomes de linhas e cores que introduziu nos passos 13 - 18 são exibidos nos cabeçalhos de linha.
    • A legenda mostra as cores que você selecionou para indicar vários estados.
    • Neste exemplo, todas as células da matriz são verde, indicando (de acordo com a legenda) que estão prontos para utilização.

Etapa 2: Configurar Células

Para configurar as células de uma matriz, acesse a ferramenta Matrizes e siga estes passos:

  1. Se a página inicial de Matrizes não for exibida, clique no botão Reset.
  2. Encontre a matriz que você deseja trabalhar e clique no link Rever abaixo do seu nome. O Sakai exibirá a tela Rever Matriz.
  3. Clique na célula desejada.
    • O Sakai dexibirá a tela Rever Celula.
    • Sakai pode ter inserido automaticamente um título para essa célula. O título padrão inclui o nome da coluna e linha que cruzam essa célula.
  4. Alterar o título da célula, se necessário. Uma entrada na caixa de Título é obrigatório.
  5. Inserir uma descrição (objectivos, etc) da célula, se necessário.
  6. Na caixa de Status Inicial, selecione Pronto ou Bloqueado para indicar se essa célula deve estar à disposição dos participantes, quando a matriz for publicado.
  7. Para aplicar um estilo a esta célula, consulte "Etapa 1: Criar o Framework," o passo 4, acima.
  8. Adicionar o seguinte, conforme necessário:
    • Formulários de estruturação de trabalho do participante do site nesta célula.
    • Instruções, fundamentos e exemplos de trabalhos relevantes para esta célula, incluindo arquivos enviados.
    • Formulários para coleta de reflexões dos participantes sobre o seu trabalho nesta célula.
    • Formulários para estruturar o feedback crítico sobre o trabalho dos participantes desse celula.
    • Formulários para avaliação do trabalho após esta célula ter sido submetido para avaliação.
    • Os nomes dos avaliadores
  9. Para salvar seu trabalho, clique em Salvar alterações. Sakai exibirá a tela Rever Matriz. Ou, para retornar a esta tela sem salvar suas entradas, clique em Cancelar.
  10. Para configurar outras células na matriz, repita os passos 3 a 9.

Etapa 3: Publicar a Matriz

A publicação de uma matriz torna disponível para uso dos participantes site. Para publicar uma matriz, acesse a ferramenta Matrizes e siga estes passos:

  1. Se a página inicial de Matrizes não for exibida, clique no botão Reset.
  2. Encontre a matriz que você deseja trabalhar e clique no link Publicar abaixo do seu nome. Sakai exibirá a tela Rever Matriz.
  3. Para publicar a matriz, clique em Continuar. Sakai publicará a matriz e exibirá a página inicial de Matrizes. Ou, para voltar à página inicial, sem a publicação da matriz, clique em Cancelar.
