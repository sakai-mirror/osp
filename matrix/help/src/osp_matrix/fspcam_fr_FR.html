Remplissage d'une matrice

Une matrice est un tableau de colonnes et de lignes qui contient une série personnalisée d'instructions, de formulaires, de fichiers et autres documents. La finalité de ces documents est de vous guider dans l'apprentissage d'activités. Ces activités se déroulent souvent sur une longue période de temps. En conséquence, vous devrez accéder à la matrice de nombreuses fois afin de la compléter.

Pour compléter une partie d'une matrice (chaque partie du tableau est appelée une « cellule »), vous y accédez via internet, vous suivez les instructions données, vous enregistrez des informations et joignez les fichiers requis, vous réfléchissez sur vos activités et vous recevez une rétroaction.

Après avoir entré les parties de votre travail dans une matrice, Sakai vous permet de créer un portfolio pour les présenter. Pour accéder à une matrice et travailler sur une ou plusieurs de ses cellules, accédez à l'outil Matrices et suivez les étapes suivantes :

  1. Si la page d'accueil des Matrices n'est pas affichée, cliquez sur le bouton Réinitialiser.
  2. Cliquez sur le nom de la matrice sur laquelle vous souhaitez travailler. Sakai affiche la matrice que vous avez choisie. Le nom de la matrice est affichée dans le coin supérieur gauche du tableau. L'intersection de chaque ligne avec une colonne forme une cellule. À l'intérieur de chaque cellule se trouvent des instructions que vous devez suivre pour prouver un certain niveau de compétence dans un domaine particulier

    Comme indiqué dans la légende sous le tableau, chaque cellule a un code couleur pour indiquer son statut.

    • Prêt : Cette cellule est prête à recevoir votre travail.
    • En attente : Vous avez soumis votre travail sur cette cellule pour évaluation.
    • Complété : Vous avez reçu une évaluation de votre travail sur cette cellule et aucun ajout n'est nécessaire.
    • Verrouillé : Cette cellule n'est pas disponible pour recevoir votre travail.
  3. Pour comprendre la finalité d'une colonne ou une ligne en particulier, utilisez votre souris pour déplacer le curseur au-dessus de celle-ci. Sakai affiche une courte description. Pour voir une description plus longue, cliquez sur le titre de la colonne ou de la ligne.
  4. Pour travailler sur une cellule, cliquez sur celle-ci.
  5. Lisez la section Instructions avec attention. Pour afficher un fichier joint listé sur l'écran, cliquez sur son nom. Sakai affiche la fenêtre Ouverture d'un fichier. Cliquez sur le bouton radio pour l'une des options : Ouvrir ou Enregistrer ; puis cliquez sur OK. Sakai affiche le fichier que vous avez choisi. Pour revenir à l'écran principal pour cette cellule, cliquez sur Retour.
  6. Selon le contenu de la cellule et le travail que vous avez effectué à ce jour, choisissez une ou plusieurs des options suivantes :
    • Complétez les formulaires listés à l'écran. Pour des informations sur le remplissage de ceux-ci, voir « Assemblage de votre travail » ci-dessous.
    • Identifiez tout item supplémentaire que vous souhaitez envoyer et joignez-le à la cellule. Pour savoir comment joindre des fichiers à une cellule voir « Assemblage de votre travail » ci-dessous.
    • Réfléchissez sur votre travail. Pour en savoir plus, voir « Réflexion sur votre travail » ci-dessous.
    • Consultez toute rétroaction formative donnée en rapport avec votre travail. Pour en savoir plus, voir « Consultation d'une rétroaction sur votre travail » ci-dessous.
    • Modifiez votre travail. Pour en savoir plus, voir « Modification de votre travail » ci-dessous.
    • Supprimez des formulaires ou du travail que vous avez joint. Pour en savoir plus, voir « Suppression des formulaires et du travail joint » ci-dessous.
    • Envoyez votre travail pour une évaluation sommative. Pour en savoir plus, voir « Envoi de votre travail pour évaluation » ci-dessous.
  7. Pour revenir à la matrice à tout moment, cliquez sur le bouton Retour à la matrice. Sakai affiche la matrice.

Assemblage de votre travail

Vous pouvez joindre des parties de votre travail sous forme de fichiers, de liens, de code HTML ou de simple texte. Pour ce faire, suivez les étapes suivantes :

  1. Si vous ne l'avez pas encore fait, téléversez les fichiers appropriés dans les Ressources. Nous vous recommandons de placer ces fichiers dans votre dossier Zone de Travail.
  2. Dans la partie Items de l'écran pour cette cellule de la matrice, cliquez sur le bouton Choisir des items supplémentaires.
  3. Suivez les instructions données. Une fois que vous avez terminé, Sakai affiche à nouveau l'écran pour cette cellule. L'item que vous avez ajouté est listé.
  4. Entrez des informations de base sur ce travail en choisissant l'une de ces options :
  5. Pour remplir un formulaire fourni pour guider votre travail dans la cellule (dans l'exemple ci-dessus, Formulaire d'Éducation Générale), cliquez sur le lien Ajouter un formulaire sous le nom. Sakai affiche le formulaire.
    6. Remplissez le formulaire et cliquez sur Ajouter. Sakai enregistre l'information et affiche de nouveau la matrice.
  6. Si vous avez déjà rempli une version de ce formulaire, elle est stockée dans les Ressources. Pour ajouter cette version du formulaire à cette cellule, cliquez sur le lien Choisir un formulaire existant sous le nom du formulaire que vous souhaitez. Quand Sakai affiche la page d'accueil des Ressources, choisissez le formulaire.
  7. Pour ajouter des items supplémentaires, répétez toutes les étapes.

Réflexion sur votre travail

Pour entrer des remarques sur votre travail, suivez les étapes suivantes :

  1. Dans la partie Réflexion de l'écran, cliquez sur le lien Entrer une remarque. Sakai affiche un formulaire.
  2. Complétez le formulaire. Une saisie dans le champ Titre est obligatoire. Dans votre institution, certains champs peuvent également être obligatoires.
  3. Une fois que vous avez complété tous les champs du formulaire, cliquez sur Ajouter. Sakai enregistre les informations et affiche de nouveau l'écran de la cellule. Le titre que vous avez entré est maintenant listé dans la partie Items.

Modification de votre travail

Avant de soumettre votre travail dans une cellule, vous pouvez modifier tout formulaire que vous avez rempli. Pour ce faire, suivez les étapes suivantes :

  1. Cliquez sur le lien Modifier sous le nom du formulaire que vous souhaitez changer. Sakai affiche le formulaire.
  2. Faites les changements que vous souhaitez et cliquez sur Mettre à jour. Sakai affiche de nouveau l'écran de la cellule. Sinon, pour revenir à cet écran sans enregistrer les changements, cliquez sur Annuler.

Suppression des formulaires et du travail joint

Sakai affiche un lien Supprimer sous le nom de chaque formulaire ou item que vous avez joint à une cellule. Pour supprimer l'un de ces items, cliquez simplement sur le lien Supprimer associé. Sakai efface l'item immédiatement.

Attention : Sakai n'affiche pas de message de confirmation de la suppression avant de supprimer l'item. Par conséquent, soyez prudent quand vous souhaitez supprimer des items.

Envoi de votre travail pour évaluation

Une fois que vous avez terminé votre travail dans une cellule, soumettez-le pour évaluation.

Attention : Une fois que vous avez envoyé votre travail pour évaluation, vous n'y avez plus accès pour effectuer des modifications. Par conséquent, vous devriez vous assurer que tout votre travail est sous sa forme finale et est prêt à être évalué et commenté avant de soumettre votre travail pour une cellule.

Pour ce faire, suivez les étapes suivantes :

  1. Cliquez sur le bouton Envoyer pour évaluation dans la partie Réflexion. Sakai affiche une écran vous demandant de confirmer que vous souhaitez envoyer votre travail pour évaluation.
  2. Pour envoyer la cellule, cliquez sur Envoyer. Sinon, pour annuler le processus d'envoi et revenir à l'écran de cette cellule, cliquez sur Annuler. Si vous avez cliqué sur Envoyer, Sakai envoie votre travail pour évaluation et affiche l'écran de cette cellule, accompagné de ce message : « Le statut est en attente et ne peut être modifié ».
  3. Retournez à la matrice en cliquant sur Retour à la matrice.
  4. Vous pouvez remarquer que la couleur de la cellule a changé. Consultez la légende sous la matrice pour identifier le statut de la cellule.

Les cellules qui ont été envoyées pour évaluation apparaissent dans la couleur pour « En attente ».

Une fois qu'une cellule a été évaluée, elle revient à la couleur « Prêt » ou passe à la couleur « Complété ». Les cellules qui reviennent à la couleur « Prêt » nécessitent un travail supplémentaire et un nouvel envoi. Les cellules qui passent à la couleur « Complété » ne sont plus disponibles pour travailler.

Consultation des rétroactions sur votre travail

Un ou plusieurs membres de votre site peuvent être assignés à l'évaluation des cellules de matrices et au dépôt de contributions sur le travail effectué par les participants sur celles-ci. Ces évaluateurs peuvent remplir un formulaire de rétroaction pour toute cellule qui n'a pas encore été envoyée. Si une rétroaction a été donnée pour une de vos cellules de matrice, vous remarquerez formulaire de rétroaction complété listé dans la cellule. Pour lire la rétroaction que vous avez reçue, suivez ces étapes :

  1. Dans la partie Rétroaction, cliquez sur le titre d'un item de la liste. Sakai affiche la rétroaction.
  2. Après avoir lu la rétroaction, fermez la fenêtre.
