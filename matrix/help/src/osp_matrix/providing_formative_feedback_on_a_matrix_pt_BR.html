Fornecer Feedback Formativo

Se você é um revisor, você pode rever as células da matriz de um participante do site a qualquer momento e oferecer feedback formativo sobre o trabalho. Você também pode retornar a uma célula revista antes, rever qualquer trabalho que foi feito lá, e fornecer feedback adicional. Para revisar e fornecer feedback sobre uma matriz, acesse a ferramenta de Matrizes e siga estes passos:

  1. Se a barra de menu de Matrizes não for exibida, clique no botão Reset.
  2. Encontre a matriz que deseja revisar e clique em seu nome. O Sakai exibirá a tela Gerenciar Matriz. A tela exibe a matriz. No canto superior esquerdo da tela, uma lista suspensa é exibida.
  3. Use a lista suspensa para selecionar o nome da pessoa cuja matriz desejada.
  4. Clique em Ir. O Sakai exibirá uma cópia de somente leitura da matriz para o usuário especificado. Ícones em uma célula indicam se o usuário anexou material. A cor de fundo de uma célula indica o seu status.
  5. Para revisar o conteúdo de uma célula, clique nela. O Sakai exibe a célula.
  6. Para rever um item anexado, clique em seu nome.
  7. Para introduzir feedback sobre o trabalho dessa célula, clique no link Adicionar Feedback na seção "Feedback". O Sakai exibirá uma tela que contém um formulário de feedback.
  8. Digite os comentários na caixa de comentários e adicione um título para o seu feedback (obrigatório) e, se desejado, uma descrição.
  9. Em "Acesso", clique em um botão para especificar se o feedback é para exibição pública, para exibir apenas a membros do site, ou nenhum.
  10. Para adicionar o seu feedback, clique em Adicionar. O Sakai exibirá o conteúdo da célula de novo. O título de seu feedback é mostrado na seção Comentários da tela.
  11. Para retornar à tela de matriz, clique em Voltar para a Matriz.
