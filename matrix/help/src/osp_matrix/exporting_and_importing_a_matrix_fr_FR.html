Exportation et Importation

Si vous possédez plus d'un site, vous pourriez vouloir utiliser la même matrice dans deux ou plusieurs d'entre eux. Pour ce faire, vous devez d'abord exporter l'assistant de saisie sur votre poste de travail sous forme de fichier compressé et ensuite l'importer sur les sites où vous avez prévu de l'utiliser. Vous pouvez modifier la matrice importée pour l'adapter aux besoins du nouveau site avant de la publier.

Exportation d'une matrice

Pour exporter une matrice sous forme de fichier compressé, accédez à l'outil Matrices et suivez les étapes suivantes :

  1. Si la page d'accueil n'est pas affichée, cliquez sur le bouton Réinitialiser.
  2. Trouvez la matrice que vous souhaitez exporter.
  3. Cliquez sur le lien Exporter sous le nom de la matrice. Sakai affiche la fenêtre Ouvrir un fichier.
  4. Suivez les instructions. Sakai crée un fichier compressé et le télécharge sur votre poste de travail.
  5. Téléversez le fichier compressé dans les Ressources.

Importation d'une matrice

Pour importer une matrice depuis votre poste de travail sous forme de fichier compressé, accédez au site sur lequel vous souhaitez l'importer et suivez les étapes suivantes :

  1. Ajoutez le fichier aux Ressources.
  2. Accédez à l'outil Matrices. Si la barre de menu des Matrices n'est pas affichée, cliquez sur le bouton Réinitialiser.
  3. Cliquez sur le bouton Importer de la barre de le menu. Sakai affiche l'écran Importer une matrice.
  4. Cliquez sur Choisir un fichier. Sakai affiche l'écran Choisir un item.
  5. Choisissez le fichier compressé pour la matrice en suivant les instructions données pour le choix d'un item. Une fois que vous avez cliqué sur Terminer, Sakai affiche de nouveau l'écran Importer une matrice. Le nom du fichier que vous avez choisi est affiché dans le champ Importer un fichier.
  6. Cliquez sur le bouton Importer l'Échafaudage. Sakai importe le fichier depuis les Ressources et affiche de nouveau la page d'accueil des Matrices. La matrice y est listée. Elle présente les mêmes noms et instructions que sur le site original. Sinon, pour revenir à la page d'accueil sans importer le fichier dans l'outil Matrices, cliquez sur Annuler. Quelque soit l'état (publié ou non publié) de la matrice sur le site original, une matrice est toujours importée à l'état non publié. Avant de publier une matrice importée, vous pouvez la modifier pour l'adapter aux besoins du nouveau site.
