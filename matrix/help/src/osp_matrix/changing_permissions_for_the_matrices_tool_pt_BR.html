Alterar Permissões

Para alterar as permissões padrão para a ferramenta Matrizes, acesse a ferramenta e siga estes passos:

  1. Se a barra de menu de Matrizes não for exibida, clique no botão Reset.
  2. Clique no botão Permissões na barra de menu. O Sakai exibirá a tela Permissões.
  3. Referindo à tabela abaixo, marque as caixas para selecionar as permissões adequadas para os usuários em cada papel.



    Permissão Descrição
    Use Os usuários neste papel podem acessar matrizes e usá-las para coletar informações e material e refletir sobre seu trabalho.
    Create Os usuários neste papel podem criar novas matrizes e importar matrizes existentes para uso próprio e/ou dos participantes do site.
    Publish Os usuários neste papel podem publicar matrizes para uso dos participantes do site.
    Export Os usuários neste papel podem exportar uma matriz para suas áreas de trabalho como um arquivo compactado.
    Review Os usuários neste papel podem rever o trabalho em andamento em matrizes (usando o menu suspenso para ir para a matriz de qualquer participante do site) e oferecer feedback formativo sobre ele.
    Evaluate Os usuários neste papel podem rever o trabalho concluído em matrizes e oferecer feedback sobre ele.


  4. Para salvar as configurações de permissões, clique em salvar. O Sakai salva as configurações e exibe a página inicial Matrizes. Ou, para voltar para a tela sem salvar as alterações, clique em cancelar.
