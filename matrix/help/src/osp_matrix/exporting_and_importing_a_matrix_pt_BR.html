Exportar e Importar

Se você possui mais de um site, você pode usar a mesma matriz em dois ou mais deles. Para isso, primeiro você exporta o assistente para sua área de trabalho como um arquivo compactado e depois importá-o para outro site (s) onde você pretende utilizá-lo. Você pode editar a matriz importada para adaptá-la às necessidades do novo site antes de publicar.

Exportar uma Matriz

Para exportar uma matriz como um arquivo compactado, acesse a ferramenta Matrizes e siga estes passos:

  1. Se a barra de menu de Matrizes não for exibida, clique no botão Reset.
  2. Encontre a matriz que você deseja exportar.
  3. Clique no link Exportar abaixo do nome da matriz. O Sakai exibirá a janela Abrir Arquivo.
  4. Siga as instruções. O Sakai cria um arquivo compactado e transfere para sua área de trabalho.
  5. Faça o upload do arquivo compactado para Repositório.

Importar uma Matriz

Para importar uma matriz da sua área de trabalho como um arquivo compactado, acesse o site ao qual você deseja importar e siga estes passos:

  1. Adicione o arquivo ao Repositório.
  2. Acesse a ferramenta Matrizes. Se a barra de menu Matrizes não for exibida, clique no botão Reset.
  3. Clique no botão Importar na barra de menu. O Sakai exibirá a tela Importar Matriz.
  4. Clique em Selecionar Arquivo. O Sakai exibirá a tela Selecionar Item.
  5. Selecione o arquivo compactado para a matriz, siga as instruções dadas para selecionar um item. Depois que você clicar em Concluir, o Sakai exibirá a tela Importar Matriz novamente. O nome do arquivo que você escolheu é exibido na caixa Importar Arquivo.
  6. Clique no botão Importar. O Sakai importará o arquivo de Repositório e exibirá novamente a página inicial Matrizes. A matriz importada é listada lá. Ele tem o mesmo nome e instruções como tinha em seu site original. Ou, para voltar à página inicial, sem importar o arquivo para da ferramenta Matrizes, clique em Cancelar. Independentemente do estado (publicado ou não publicado) da matriz no site original, é sempre importada no estado não publicada. Antes de publicar a matriz importada, você pode alterá-la para se adequar às exigências do novo site.
