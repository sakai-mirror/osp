Création d'une matrice

La création d'une matrice se déroule en plusieurs étapes :

  1. Création d'une structure pour la matrice.
  2. Préparation des cellules.
  3. Publication de la matrice.

Étape 1 : Création de la structure

Pour créer la structure d'une matrice, accédez à l'outil Matrices et suivez les étapes suivantes :

  1. Si la page d'accueil des Matrices n'est pas affichée, cliquez sur le bouton Réinitialiser.
  2. Dans la barre de menu des Matrices en haut de l'écran, cliquez sur le bouton Ajouter. Sakai affiche l'écran Ajout d'une matrice.
  3. Dans les champs correspondants, entrez un titre et une description de votre choix pour la matrice. Seul le champ Titre est obligatoire. Attention : Choisissez le titre avec attention. Une fois la matrice publiée, vous ne pouvez plus changer son titre.
  4. Choisissez l'une des options suivantes :
    • Pour appliquer le style par défaut de Sakai à cette matrice, passez cette étape et allez directement à l'étape 6.
    • Pour appliquer un autre style à cette matrice, cliquez sur Sélectionner un style. Sakai affiche l'écran Gestion des Styles. Cet écran présente une liste des styles utilisables pour vos matrices.
  5. Cliquez sur le lien Sélectionner pour le style que vous souhaitez appliquer à cette matrice. Sakai affiche à nouveau l'écran Ajout d'une matrice. Le nom du style que vous avez choisi est affiché dans le champ Style. Sinon, pour revenir à cet écran sans sélectionner de style, cliquez sur Retour.
  6. Pour ajouter une colonne, cliquez sur Ajout d'une colonne. Sakai affiche l'écran Édition des Colonnes de l'Échafaudage.
  7. Dans le champ Nom, entrez le nom qui sera affiché dans l'en-tête de colonne. Gardez à l'esprit que des noms courts pour les colonnes permettent une meilleure utilisation de l'espace disponible à l'écran.
  8. Pour choisir une couleur de fond pour cette colonne différente de celle par défaut (gris), cliquez sur l'icône de palette pour le champ Couleur du fond. Quand Sakai affiche les choix de couleur, cliquez sur la couleur de fond que vous souhaitez.
  9. Pour choisir une couleur pour le texte dans cette colonne différente de celle par défaut (noir), cliquez sur l'icône palette pour le champ Couleur de la police. Quand Sakai affiche les choix de couleur, cliquez sur la couleur de police que vous souhaitez.
  10. Pour enregistrer votre travail sur cette colonne, cliquez sur Mettre à jour. Sakai affiche l'écran Ajout d'une matrice, comprenant des informations sur la colonne que vous venez de créer. Sinon, pour revenir à l'écran Ajouter une matrice sans ajouter cette colonne, cliquez sur Annuler.
  11. Répétez les étapes 6 à 10 autant de fois que nécessaire pour créer les autres colonnes de votre matrice. Attention : Choisissez avec attention le nombre de colonnes dans la matrice. Une fois publiée, vous ne pouvez pas changer le nombre de colonnes qu'elle comprend.
  12. Pour changer la terminologie pour autre chose que le mot « Colonne », entrez le terme que vous désirez dans le champ Libellé de colonne.
  13. Pour ajouter une ligne, cliquez sur Ajouter une ligne. Sakai affiche l'écran Édition des Lignes de l'Échafaudage.
  14. Dans le champ Nom, entrez un nom qui sera affiché dans l'en-tête de la ligne.
  15. Pour choisir une couleur de fond pour cette ligne différente de celle par défaut (gris), cliquez sur l'icône de palette pour le champ Couleur du fond. Quand Sakai affiche les choix de couleur, cliquez sur la couleur de fond que vous souhaitez.
  16. Pour sélectionner une couleur pour le texte dans cette ligne différente de celle par défaut (noir), cliquez sur l'icône palette pour le champ Couleur de la police. Quand Sakai affiche les choix de couleur, cliquez sur la couleur de police que vous souhaitez.
  17. Pour enregistrer votre travail sur cette ligne, cliquez sur Mettre à jour. Sakai affiche l'écran Ajout d'une matrice, comprenant des informations sur la ligne que vous venez de créer. Sinon, pour revenir à l'écran Ajouter une matrice sans ajouter cette ligne, cliquez sur Annuler.
  18. Répétez les étapes 13 à 17 autant de fois que nécessaire pour créer les autres lignes de votre matrice. Attention : Choisissez avec attention le nombre de lignes dans la matrice. Une fois publiée, vous ne pouvez pas changer le nombre de lignes qu'elle comprend.
  19. Pour changer la terminologie pour autre chose que le mot « Ligne », entrez le terme que vous désirez dans le champ Libellé de ligne.
  20. Dans la partie Progression de la matrice, cliquez sur le bouton radio approprié pour choisir la façon dont vous souhaitez que les participants travaillent sur les cellules de la matrice.
  21. Dans la partie Couleurs de statut de la matrice, choisissez les couleurs pour indiquer le statut de chaque cellule de la matrice.



    Statut Signification
    Prêt Cette cellule est prête à recevoir le travail d'un participant.
    En attente Un participant a soumis cette cellule pour évaluation.
    Complété Cette cellule a été évaluée et validée.
    Verrouillé Cette cellule n'est pas disponible pour la réception du travail d'un participant pour le moment.


  22. Pour enregistrer votre travail, cliquez sur Enregistrer les modifications. Sinon, pour quitter sans ajouter de nouvelle matrice, cliquez sur Annuler.

    Si vous avez cliqué sur Enregistrer les modifications, Sakai enregistre vos modifications et affiche la nouvelle matrice dans l'écran Modification d'une matrice.

    À noter :
    • Le titre de matrice que vous avez renseigné dans l'étape 3 est affiché dans le coin supérieur gauche du tableau. Si vous avez entré une description dans cette étape, elle est affichée au-dessus du tableau.
    • Les noms de colonne, couleurs de fond et de texte que vous avez saisis dans les étapes 6 à 11 sont affichés dans les en-têtes de colonne.
    • Les noms de ligne et les couleurs que vous avez saisis dans les étapes 13 à 18 sont affichés dans les en-têtes de ligne.
    • La légende présente les couleurs que vous avez choisies pour indiquer les différents états.
    • Dans cet exemple, toutes les cellules de la matrice sont vertes, indiquant (d'après la légende) qu'elles sont prêtes à être utilisées.

Étape 2 : Préparation des cellules

Pour préparer les cellules d'une matrice, accédez à l'outil Matrices et suivez les étapes suivantes :

  1. Si la page d'accueil des Matrices n'est pas affichée, cliquez sur le bouton Réinitialiser.
  2. Trouvez la matrice sur laquelle vous souhaitez travailler et cliquez sur le lien Réviser sous son nom. Sakai affiche l'écran Réviser une matrice.
  3. Cliquez sur la cellule désirée.
    • Sakai affiche l'écran Modification d'une cellule.
    • Sakai a automatiquement entré un titre pour cette cellule. Le titre par défaut inclut à la fois le nom de la colonne et de la ligne qui sont à l'intersection de cette cellule.
  4. Si besoin, changez le titre de la cellule. Une saisie dans le champ Titre est obligatoire.
  5. Si besoin, entrez une description (objectifs, etc.) de la cellule.
  6. Dans le champ Statut initial, choisissez Prêt ou Verrouillé pour indiquer si cette cellule doit être disponible pour les participants dès la publication de la matrice.
  7. Pour appliquer un style à cette cellule, consultez « Étape 1 : Création de la structure », étape 4 ci-dessus.
  8. Ajoutez si besoin :
    • Des formulaires pour structurer le travail des participants pour cette cellule.
    • Des instructions, des raisons et des exemples de travail en rapport avec cette cellule, comprenant des fichiers téléversés.
    • Des formulaires pour collecter les remarques des participants sur leur travail pour cette cellule.
    • Des formulaires pour structurer le commentaire de l'évaluateur sur le travail du participant pour cette cellule.
    • Des formulaires pour l'évaluation du travail après que cette cellule ait été soumise pour évaluation.
    • Les noms des évaluateurs.
  9. Pour enregistrer votre travail, cliquez sur Enregistrer les modifications. Sakai affiche l'écran Modification d'une matrice. Sinon, pour retourner à cet écran sans enregistrez vos saisies, cliquez sur Annuler.
  10. Pour préparer d'autres cellules dans la matrice, répétez les étapes 3 à 9.

Étape 3 : Publication d'une matrice

La publication d'une matrice la rend disponible pour tous les participants du site. Pour publier une matrice, allez à l'outil Matrices et suivez ces étapes :

  1. Si la page d'accueil des Matrices n'est pas affichée, cliquez sur le bouton Réinitialiser.
  2. Trouvez la matrice sur laquelle vous souhaitez travailler et cliquez sur le lien Publier sous son nom. Sakai affiche l'écran Modification d'une matrice.
  3. Pour publier la matrice, cliquez sur Continuer. Sakai publie la matrice et affiche la page d'accueil des Matrices. Sinon, pour revenir à la page d'accueil sans publier la matrice, cliquez sur Annuler.
