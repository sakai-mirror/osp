Requête pour publier une mise en page à toutes les communautés

Si vous avez créé une mise en page qui peut s'appliquer de façon générale, vous pourriez vouloir la rendre utilisable pour toutes les communautés du bureau virtuel. Seuls les administrateurs système peuvent publier une mise en page de façon globale (c'est-à-dire pour toutes les communautés Sakai).

Pour demander à ce qu'une mise en page soit publiée à toutes les communautés, accédez à l'outil Mises en page des portfolios et suivez les étapes suivantes :

  1. Si la page d'accueil des Mises en page des portfolios n'est pas affichée, cliquez sur le bouton Réinitialiser.
  2. Trouvez la mise en page que vous souhaitez publier pour toutes les communautés.
  3. Cliquez sur le lien Suggérer pour une publication globale sous le nom de votre mise en page. Sakai envoie la mise en page pour validation et affiche de nouveau la page d'accueil des mises en page des portfolios.
