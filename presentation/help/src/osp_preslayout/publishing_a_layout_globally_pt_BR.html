Publicar um Layout Globalmente

Se você é um administrador do sistema, você pode publicar um layout globalmente (isto é, para todos os sites em uma instância SAKAI). Para isso, faça o login e siga estes passos:

  1. Clique na pasta Administração no topo da tela.
  2. Selecione o site PortfolioAdmin.
  3. Acesse a ferramenta Portfólio Layouts. O Sakai exibirá a página inidical de Portfólio Layouts.
  4. Encontre o layout que você deseja publicar e clique no link Publica abaixo do seu nome. O Sakai torna o layout disponível para todos os sites.
