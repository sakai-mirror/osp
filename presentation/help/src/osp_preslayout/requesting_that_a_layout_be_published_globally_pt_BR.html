Solicitar que um Layout Seja Publicado Globalmente

Se você criou um layout que tem aplicabilidade geral, você pode querer torná-lo disponível para uso em todos os sites em sua CLE. No entanto, apenas o administrador do sistema Sakai pode publicar um layout globalmente (isto é, em todos os sites do Sakai).

Para solicitar que um layout seja publicado a nível global, acesse a ferramenta Portfólios Layouts e siga estes passos:

  1. Se a página inicial de Portfólio Layouts não for exibida, clique no botão Reset.
  2. Encontre o layout que você deseja sugerir para publicação global.
  3. Clique no link Sugerir para Publicação Global abaixo do nome do layout. O Sakai submete o layout para consideração de publicaçãor global e exibe a página inicial de Portfólio Layouts.
