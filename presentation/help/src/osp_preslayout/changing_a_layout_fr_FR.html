Modification d'une mise en page

Vous pouvez modifier le titre et la description des mises en page. Vous pouvez également joindre différents fichiers. Pour modifier une mise en page, accéder à l'outil mises en page des portfolios et suivez les étapes suivantes :

  1. Si la page d'accueil des Mises en page des Portfolios n'est pas affichée, cliquez sur le bouton Réinitialiser.
  2. Cliquez sur le lien Modifier sous le nom de la mise en page à modifier. Sakai affiche l'écran Modification d'une mise en page.
  3. Modifiez le contenu des champs Nom affiché et Description comme vous le souhaitez.
  4. Pour modifier la mise en page ou le fichier d'aperçu, cliquez sur le lien Choisir un fichier pour le champ approprié. Quand Sakai affiche l'écran Choisir un item, sélectionnez le fichier à modifier en suivant les instructions sur le choix d'un item. Une fois que vous avez cliqué sur Terminer, Sakai affiche à nouveau l'écran Ajouter une mise en page. Le nom du fichier choisi est affiché dans le champ.
  5. Sélectionnez le fichier XHTML que vous désirez en suivant les instructions pour choisir un item. Une fois que vous avez cliqué sur Terminer, Sakai affiche à nouveau l'écran Ajouter une mise en page. Le nom du fichier choisi est affiché dans le champ.
  6. Cliquez sur le bouton Enregistrer les modifications. Sakai effectue les modifications et affiche la liste sur la page d'accueil des mises en page des portfolios. Pour revenir à cette page sans modifier la mise en page, cliquez sur Annuler.
