Deletar um Layout

Você pode deletar um portfólio layout apenas se ele não estiver sendo usado em qualquer um dos portfolios no site. Se o layout estiver em uso, você deve deletar todos os portfólios que estiverem em uso antes de deletar o layout.

Para deletar um portfólio layout, acesse a ferramenta Portfólio Layouts e siga os seguintes passos:

  1. Se a página inicial de Portfólio Layouts não for exibida, clique no botão Reset.
  2. Encontre o layout que você deseja remover e clique no link Excluir abaixo do seu nome. O Sakai exibirá a janela para confimar a exclusão.
  3. Para deletar o layout, clique em OK. O Sakai deletará o layout e exibirá a página inicial de Portfólio Layouts. Ou, para returnar a página inicial sem deletar esse layout, clique em Cancelar.
