Modification des permissions

Pour changer les permissions des utilisateurs d'un ou plusieurs rôles, accédez à l'outil Mises en page des portfolios et suivez les étapes suivantes :

  1. Si la barre de menu des mises en page des portfolios n'est pas affichée, cliquez sur le bouton Réinitialiser.
  2. Cliquez sur le bouton Permissions dans la barre de menu. Sakai affiche l'écran Permissions.
  3. En vous reportant au tableau ci-dessous, cochez les cases pour donner les permissions appropriées pour les utilisateurs de chaque rôle.



    Permission Description
    Publication Les utilisateurs de ce rôle peuvent publier des mises en page de portfolio utilisables par les participants du site.
    Suppression Les utilisateurs de ce rôle peuvent supprimer des mises en page de portfolio.
    Création Les utilisateurs de ce rôle peuvent ajouter de nouvelles mises en page de portfolio.
    Modification Les utilisateurs de ce rôle peuvent modifier les mises en page de portfolio.
    Suggestion de publication globale Les utilisateurs de ce rôle peuvent suggérer qu'une mise en page de portfolio soit publiée globalement (c'est-à-dire pour toutes les communautés Sakai). Seul l'administrateur système peut publier globalement des mises en page.


  4. Pour enregistrer les réglages de permissions, cliquez sur le bouton Enregistrer. Sakai enregistre les paramètres et affiche la page d'accueil des mises en page des portfolios. Pour revenir à la page d'accueil sans enregistrer les modifications, cliquez sur Annuler.
