Adicionar um Layout

Se você é um administrado do sistema, será avisado que qualquer layout que você adicionar ao site PortfolioAdmin torna-se automaticamente disponível para todos os sites em sua instância SAKAI.

Para adicionar um portfólio layout ao site, siga esses passos:

  1. Adicione arquivos para esse layout em Repositório.
  2. Acesse a ferramenta Portfólio Layouts.
  3. Se a página inicial de Portfólio Layouts não for exibida, clique no botão Reset.
  4. Na barra de menu do Portfolio Layouts perto do topo da tela, clique no botão Adicionar. O Sakai exibirá a tela Adicionar Layout.
  5. Na caixa Nome de Exibição, digite um nome significativo para o layout. Esse nome será exibido aos participantes do site quando eles criarem portfolios. Essa entrada é obrigatório. Se desejar, digite uma descrição para esse layout na caixa Descrição.
  6. Clique no link Selecionar Arquivo na caixa Arquivo Layout XHTML. O Sakai exibirá a tela Selecionar Item.
  7. Selecione o arquivo XHTML que você precisa seguindo as instruções para selecionar um item. Depois clique em Concluir, o Sakai exibirá a tela Adicionar Layout novamente. O nome do arquivo que você selecionou será exibido na caixa.
  8. Para fazer uma pré-visualização do layout disponível, clique no link Selecionar Arquivo para a caixa de Visualização de Imagem. O Sakai exibe a tela Selecionar Item. Proceder como no passo 7, acima.
  9. Para adicionar esse layout no site, clique no botão Adicionar Layout. O Sakai exibirá a página inicial de Portfolio Layouts. O novo layout será exibido na lista. Ou, para retornar a página sem adicionar o novo layout, clique em Cancelar.
