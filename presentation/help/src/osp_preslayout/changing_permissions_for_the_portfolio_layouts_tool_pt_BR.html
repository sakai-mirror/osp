Alterar Permissões

Para alterar permissões para os usuários em um ou mais papéis, acesse a ferramenta Portfólio Layouts e siga os seguintes passos:

  1. Se a página inicial de Portfólio Layouts não for exibida, clique no botão Reset.
  2. Clique no botão Permissões na barra de menu. O Sakai exibirá a tela de Permissões.
  3. Referindo-se a tabela abaixo, marque as caixas para selecionar as permissões apropriadas para todos os usuários em cada papel.



    Permissão Descrição
    Publicar Usuários nesse papel podem publicar portfólio layouts para serem usados pelos participantes do site.
    Deletar Usuários nesse papel podem deletar portfólio layouts.
    Criar Usuários nesse papel podem adicionar novo portfólio layouts.
    Editar Usuários nesse papel podem alterar portfólio layouts.
    Sugerir Publicação Global Usuários nesse papel podem sugerir que um portfolio layout seja publicado globalmente (isto é, disponível em todos os sites do Sakai). Apenas o administrador do sistema pode publicar layouts globalmente.


  4. Para salvar as configurações das permissões, clique no botão salvar. O Sakai salvará as configurações e exibirá a página inicial de Portfólio Layouts. Ou, para returnar a página inicial sem salvar suas alterações, clique em cancelar.
