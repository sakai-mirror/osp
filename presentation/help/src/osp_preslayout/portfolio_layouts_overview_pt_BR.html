Portfólio Layouts: Resumo

Portfólio layouts determina o arranjo dos itens em páginas portfólio criadas por participantes do site. Cada participante pode projetar um portfólio por escolher um layout e usá-lo para exibir os itens selecionados do Repositório.

O Sakai contém um portfólio layout padrão que está disponível para todos os participantes do site. Se você é um organizador do site ou tem permissão adequada, você pode usar a ferramenta Portfólio Layouts para adicionar mais portfólio layouts ao site para uso de todos os participantes. Depois de criar um layout, você pode alterar e excluí-lo, conforme necessário. Você também pode usar essa ferramenta para adicionar visualização de imagens para ajudar os participantes a escolher layouts apropriados para seus portfólios e para alterar as permissões de uso de todos os usuários em qualquer papel.

Todos os portfólios layouts são gerados por arquivos XHTML, e todas as visualizações de layout são .jpg. Você deve armazenar esses arquivos no Repositório antes de utilizar a ferramenta Portfólio Layouts para acessá-los.

A ferramenta Portfólio Layout não permite mudar os modelos de portfólio.

A ferramenta Portfólio Layouts geralmente está localizada na categoria Design & Publicação. Para acessar a ferramenta, clique no botão Iniciar Portfólio Layouts. O Sakai exibe a página inicial de Portfólio Layouts.

Esta página contém uma lista de todos os portfólios layouts no site. Se a lista é maior que uma página, use os botões da barra de menu do Portfólio Layouts para exibir outras páginas na lista.

  • Para exibir a primeira página da lista, clique em Primeiro.
  • Para exibir a página anterior da lista, clique em Anterior.
  • Para exibir a página seguinte da lista, clique em Avançar.
  • Para exibir a última página da lista, clique em Última.
