Alterar um Layout

Você pode alterar o título e a descrição para um layout. Você também pode anexar diferentes arquivos. Para alterar um layout, acesse a ferramenta Portfólio Layouts e siga os seguintes passos:

  1. Se a página inicial de Portfólio Layouts não for exibida, clique no botão Reset.
  2. Clique no link Revisar abaixa do nome do layout que você deseja alterar. O Sakai exibirá a tela Revisar Layout.
  3. Altere as entradas nas caixas Nome de Exibição e Descrição, como necessário.
  4. Para alterar o layout ou visualizar o arquivo, clique no link Selecionar Arquivo na caixa apropriada. Quando o exibir a tela Selecionar Item, selecione o arquivo que você precisa seguindo as instruções para selecionar um item. Depois clique em Concluir, o Sakai exibirá a tela Adicionar Layout novamente. O nome do arquivo que você selecionou é mostrado na caixa.
  5. Selecione o arquivo XHTML que você precisa seguindo as instruções para selecionar um item. Depois clique em Concluir, o Sakai exibirá a tela Adicionar Layout novamente. O nome do arquivo que você selecionou será mostrado na caixa.
  6. Clique no botão Salvar Alterações. O Sakai fará suas alterações e exibirá a lista da página inicial de Portfólio Layouts. Ou, para returnar a página inicial sem salvar as alterações do layout, clique em Cancelar.
