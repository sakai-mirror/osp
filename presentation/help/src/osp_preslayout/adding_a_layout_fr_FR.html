Ajout d'une mise en page

Si vous êtes administrateur système, soyez averti que toute mise en page que vous ajoutez à l'administration des portfolios devient automatiquement disponible pour tous les sites de votre instance Sakai.

Pour ajouter une mise en page de portfolio à votre site, suivez les étapes suivantes :

  1. Ajoutez les fichiers pour cette mise en page dans Ressources.
  2. Accédez à l'outil Mises en page des portfolios.
  3. Si la page d'accueil des mises en page des portfolios n'est pas affichée, cliquez sur le bouton Réinitialiser.
  4. Dans la barre de menu en haut de l'écran, cliquez sur le bouton Ajouter. Sakai affiche l'écran Ajouter une mise en page.
  5. Dans le champ Nom affiché, entrez un nom significatif pour la mise en page. Ce nom sera affiché aux participants du site lorsqu'ils créeront des portfolios. Ce champ est obligatoire. Si vous le souhaitez, entrez une description de cette mise en page dans le champ Description.
  6. Cliquez sur le lien Choisir un fichier pour le champ Mise en page XHTML. Sakai affiche l'écran Choisir un item.
  7. Choisissez le fichier XHTML souhaité en suivant les instructions pour choisir un item. Une fois que vous avez cliqué sur Terminer, Sakai retourne à l'écran Ajouter une mise en page. Le nom du fichier que vous avez choisi s'affiche dans le champ.
  8. Pour réaliser un aperçu de la mise en page, cliquez sur le lien Choisir un fichier pour le champ Image d'aperçu. Sakai affiche l'écran Choisir un item. Procédez de la même façon que pour l'étape 7 ci-dessus.
  9. Pour ajouter cette mise en page au site, cliquez sur le bouton Ajouter la mise en page. Sakai affiche la page d'accueil mises en page des portfolios. La nouvelle mise en page est affichée dans la liste. Pour revenir à cette page sans ajouter la nouvelle mise en page, cliquez sur Annuler.
