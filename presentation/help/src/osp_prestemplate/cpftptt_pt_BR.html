Alterar Permissões

Para alterar as permissões para usuários de um ou mais papéis, acesse a ferramenta Modelos de Portfólio Templates e siga esses passos:

  1. Se a página inicial de Modelos de Portfólio não for exibida, clique no botão Reset.
  2. Clique no botão Permissões na barra de menu. O Sakai exibirá a tela de Permissões.
  3. Referindo-se a tabela abaixo, marque as caixas para selecionar as permissões apropriadas para todos os usuários em cada papel.



    Permissão Descrição
    copiar Usuários nesse papel podem criar novos modelos por copiarem modelos existentes.
    publicar Usuários nesse papel podem publicar modelos para serem usados pelos participantes do site.
    deletar Usuários nesse papel podem deletar modelos.
    criar Usuários nesse papel podem usar a ferramenta Modelos de Portfólio para criar e importar modelos.
    editar UUsuários nesse papel podem alterar modelos.
    exportar Usuários nesse papel podem exportar um modelo para sua área de trabalho como arquivo compactado.


  4. Para salvar as configurações das permissões, clique no botão salvar. O Sakai salva as configurações e exibe a página inicial de Modelos de Portfólio. Ou, para returnar a página inicial sem salvar suas alterações, clique em cancelar.
