Outil Ressources

Si vous êtes un gestionnaire de site, l'outil de modèles portfolio vous permet d'ajouter des modèles qui définissent le contenu, style et la disposition des portfolios créés par les participants de votre site. Chaque modèle est basé sur un fichier .xsl et peut inclure d'autres fichiers, tels des .css, .xsd et des javascripts. Vous devez téléverser ces fichiers aux ressources pour les rendre disponibles pour l'outil de modèles portfolio.

Après l'ajout d'un modèle, vous avez la possibilité d'utiliser cet outil pour chacune des activités suivantes :

  • Le modifier ou l'effacer.
  • Le publier sur le site pour que les participants puissent l'utiliser.
  • Créer un nouveau modèle à partir d'une copie retravaillée de ce modèle.
  • Utiliser le modèle sur un autre site en l'exportant à partir de votre site et en l'important sur l'autre.
  • Changer les permissions de l'outil de modèles portfolio pour tous les utilisateurs et pour tous les rôles.
