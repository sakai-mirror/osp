Suppression d'un modèle

Vous avez la possibilité d'effacer un modèle uniquement si il n'est utilisé dans aucun portfolio (publié ou non). Si le modèle est utilisé, vous devez effacer le(s) portfolio(s) basé(s) sur lui avant de pouvoir effacer le modèle lui-même.

Pour effacer un modèle, accédez à l'outil des modèles portfolio et suivez ces étapes :

  1. Si la page d'accueil des modèles portfolio n'est pas affichée, cliquez sur le bouton "Réinitialiser".
  2. Trouvez le modèle que vous voulez enlever et cliquez sur le lien "Effacer" en dessous de son nom. Sakai affiche une fenêtre de confirmation.
  3. Pour effacer le modèle, cliquez sur "Confirmer". Sakai efface le modèle et affiche la page d'accueil des modèles portfolio. Sinon, pour retourner à la page d'accueil sans effacer le modèle, cliquez sur "Annuler".
