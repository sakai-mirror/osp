Exportation et importation

Si vous possédez plus d'un site, vous pourriez vouloir utiliser le même modèle dans l'un ou l'autre de ces sites. Pour ce faire, vous devez en premier lieu exporter le modèle sur votre bureau en tant que fichier compressé et ensuite l'importer dans l'autre site sur lequel vous prévoyez de l'utiliser.

Exporter un modèle

Pour exporter un modèle en tant que fichier compressé, accédez à l'outil de modèle portfolio sur le site approprié et suivez ces étapes :

  1. Si la page d'accueil des modèles portfolio n'est pas affichée, cliquez sur le bouton "Réinitialiser".
  2. Trouvez le modèle avec lequel vous souhaitez travailler et cliquez sur le lien "Exporter" en dessous de son nom. Sakai affiche la fenêtre d'ouverture de fichier.
  3. Suivez les instructions.

Importer un modèle

Pour importer un modèle compressé à partir de votre bureau vers un site Sakai, accédez au site sur lequel vous voulez l'importer et suivez ces étapes :

  1. Ajoutez le fichier aux ressources.
  2. Accédez à l'outil de modèle portfolio. Si la page d'accueil n'est pas affichée, cliquez sur le bouton "Réinitialiser".
  3. Cliquez sur le lien d'importation sur la barre de menu des modèles portfolio près du sommet de la page. Sakai affiche l'écran d'importation de modèle.
  4. Cliquez sur le lien "Sélectionner un fichier". Sakai affiche l'écran de sélection.
  5. Choisissez le fichier compressé pour le modèle en suivant les instructions pour sélectionner un item. Après que vous ayez cliqué sur "Terminer", Sakai affiche l'écran d'importation de modèle à nouveau. Le nom du fichier que vous avez choisi est affiché dans la ligne du fichier de modèle.
  6. Pour importer le fichier, cliquez sur "Importer le modèle". Sakai importe le fichier et affiche la page d'accueil des modèles portfolio. Le modèle importé est listé ici. Sinon, pour revenir à la page d'accueil sans importer le fichier, cliquez sur "Annuler".
