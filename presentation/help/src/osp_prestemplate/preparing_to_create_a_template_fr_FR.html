Préparation à la création d'un modèle

Avant de créer un modèle, nous recommendons que vous vous familiarisez avec les formes, dispositions et styles que votre institution a créé. Ensuite, prenez en considération les questions suivantes et agissez de manière appropriée et avec précaution.

  • Quels sont les objectifs du type de portfolios qui seront basés sur ce modèle ?
  • Quel contenu doit être inclu pour accomplir ces objectifs ?
  • Dans quel ordre le contenu doit-il être présenté ?
  • Si votre institution a des dispositions de portfolio existantes, laquelle de ces dispositions est la meilleure pour les portfolios construits sur ce modèle ? Téléversez le fichier .xsl approprié aux ressources. Si vous n'avez pas un fichier de disposition approprié, téléversez passthrough.xsl aux ressources. Ce fichier est disponible dans \webapps\osp-portal\WEB-INF\transform et dans la section Downloads/OSP de rSmart RSN.
  • Les utilisateurs ont-ils la nécessité d'entrer des informations additionnelles au portfolio telles un titre, une en-tête ou des images ? Les utilisateurs ont-ils besoin de sélectionner un seul niveau d'une matrice pour l'inclure dans le portfolio ? Si la réponse à une seule des précédentes question est Oui, dans quel fichier ces options sont-elles disponibles ? Téléversez le fichier .xsd approprié aux ressources et faites attention à son noeud racine.
  • Votre institution a un "style" par défaut qui inclue un standart de polices, couleurs, et d'autres choses. Si votre institution a créé d'autres styles, lequel est le meilleur pour les portfolios basés sur ce modèle ? Téléversez le fichier .css approprié aux ressources. Faites attention que le nom de ce fichier soit utilisé dans le fichier de disposition (.xsl) que vous avez choisi.
  • De quels autres fichiers avez-vous besoin d'utiliser dans ce modèle ? Vous pouvez inclure des images, du son, de la video, du javascript, et d'autres types de fichiers. Téléversez les fichiers appropriés aux ressources et prenez garde que le nom de chacun d'entre eux soit présent dans le fichier de disposition (.xsl) que vous avez choisi.
