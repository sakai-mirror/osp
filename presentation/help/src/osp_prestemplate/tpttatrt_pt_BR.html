Ferramenta Repositório

Se você é o organizador do site, a ferramenta Modelos de Portfólio permite que você adicione modelos que define o conteúdo, estilo e layout dos portfólios criados para os participantes do seu site. Cada modelo é baseado em um arquivo .xsl e pode incluir outros arquivos, como arquivos .css, .xsd e javascript. Você pode fazer o upload desses arquivos no Repositório para disponibilizá-los para a ferramenta Modelos de Portfólio.

Depois de adicionar um modelo, você pode usar essa ferramenta para realizar uma das atividades a seguir.

  • Alterar e excluir.
  • Publicar para ser usado pelos participantes do site.
  • Criar novo modelo por copiar o modelo que você adicionou e alterar a cópia.
  • Usar um modelo criado em um site em outro site por exportar do primeiro site e importar no outro.
  • Alterar as permissões da ferramenta Modelos de Portfólio para todos os usuários em qualquer papel.
