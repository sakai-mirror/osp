Création d'un modèle

Créer un modèle implique plusieurs étapes :

  1. Ajout du nom du modèle à Sakai.
  2. Spécification de la disposition pour le modèle.
  3. Ajout des formes requises et autres items.
  4. Sélection des fichiers de support.
  5. Si le modèle est construit sur passthrough.xsl, création d'un fichier de transformation .xsl et ajout au modèle.

Etape 1 : Ajout du nom de modèle

Pour donner un nom au modèle, accédez à l'outil des modèles portfolio et suivez ces étapes :

  1. Si la page d'accueil des modèles portfolio n'est pas affichée, cliquez sur le bouton "Réinitialiser".
  2. Dans la barre de menu des modèles portfolio proche du sommet de la page, cliquez sur le bouton "Ajouter". Sakai affiche l'écran d'ajout de modèle portfolio, étape 1 sur 4.
  3. Ajoutez un nom pour cette disposition dans la case de saisie du nom. Si vous le souhaitez, ajoutez une description dans la case de saisie de description. Dans "Affichage de la navigation portfolio", spécifiez si les portfolios basés sur ce modèle s'afficheront dans une nouvelle fenêtre internet, ou à partir de la fenêtre d'où on accède.



    Option Résultat
    Oui Les portfolios basés sur ce modèle s'afficheront dans la fenêtre à partir de laquelle on accède à eux.
    Non Les portfolios basés sur ce modèle s'afficheront dans une nouvelle fenêtre du navigateur.


  4. Pour procéder, cliquez sur "Continuer". Sakai affiche l'outil de construction de modèle (étape 2 sur 4, voir ci-dessous). Sinon, pour revenir à la page d'accueil des modèles portfolio sans valider aucune de ces informations, cliquez sur "Annuler".

Etape 2 : spécification de la disposition

Si vous avez cliqué sur "Continuer", Sakai affiche la page de construction de modèle, étape 2 sur 4.

Pour spécifier la disposition pour ce modèle, suivez ces étapes :

  1. A la droite de la zone du modèle basique, cliquez sur "Sélectionner un fichier". Sakai affiche l'item de sélection.
  2. Selectionnez le fichier .xsl nécessaire (si vous n'avez pas un fichier .xsl spécifique pour le modèle de portfolio, selectionnez passthrough.xsl). Pour sélectionner le fichier, suivez les instructions pour choisir un item. Après avoir cliqué sur "Validation", Sakai affiche l'outil de construction de modèle (étape 2 sur 4) de nouveau. Le nom du fichier que vous avez choisi est affiché dans la zone du modèle basique d'émission.
  3. Vous avez la possibilité de régler des options d'émission en suivant ces étapes :
  4. Cliquez sur le lien de sélection de fichier à la droite de zone des options.
  5. Selectionnez le fichier .xsd approprié en suivant les instructions de sélection d'un item. Après que vous ayez cliqué sur "Continuer", Sakai affiche la page du modèle de construction (étape 2 sur 4). Le nom du fichier sélectionné apparaît dans la zone d'option d'émission.
  6. Utilisez le menu déroulant dans la zone d'option d'émission des fichiers pour choisir le noeud racine désiré à partir du fichier que vous avez spécifié.
  7. Choisissez une des options suivantes :
    • Pour continuer à construire le modèle, cliquez "Continuer". Sakai affiche la page listant le contenu (étape 3 sur 4).
    • Pour arrêter de travailler sur ce modèle et sauvegarder votre travail, cliquez sur "Finir". Sakai affiche la page d'accueil des modèles portfolio. Votre nouveau modèle est listé sur cette page. Vous avez la possibilité de reprendre votre travail plus tard.
    • Pour revenir à la page d'ajout de modèle portfolio (étape 1 sur 4), cliquez sur "Retour".
    • Pour quitter votre travail sans sauvegarder, cliquez sur "Annuler". Sakai affiche la page d'accueil des modèles portfolio.

Etape 3 : Ajout des formes requises et autres items

Si vous avez choisi de continuer, Sakai affiche la page affichant le contenu (étape 3 sur 4).

Cette page vous permet d'identifier les formes, matrices, fichiers téléversés, liens URL et autres contenus pour portfolio qui utilisent ce modèle. Pour sélectionner ce matériel, suivez ces étapes :

  1. Dans le champ de saisie, utilisez le menu déroulant pour sélectionner le type d'item que le participant du site doit ajouter. Une sélection est ici requise. La liste contient les noms de tous les élements sur le site, à côté avec deux autres choix : matrice et et fichier téléversé. Si le participant du site doit :
    • compléter et attacher une forme spécifique, sélectionnez le nom de cette forme ici.
    • attacher une matrice, sélectionnez "matrice".
    • attacher n'importe quel type de ressource (simple texte, URL, fichier, etc...) selectionnez "fichier téléversé".
  2. En vous réferrant à la table ci-dessous, entrez un nom approprié dans la case du nom. Cette entrée est requise.



    Item Entrée
    Forme Le noeud racine dans ce fichier
    Fichier téléversé Le nom par lequel le fichier .xsl identifie ce fichier, si le modèle est basé sur passthrough.xsl ; un seul mot -n'importe- par lequel le fichier .xsl final identifie ce fichier.
    Matrice Le nom par lequel le fichier .xsl identifie cette matrice, si le modèle est basé sur passthrough.xsl ; un seul mot -n'importe- par lequel le fichier .xsl final identifie ce fichier.


  3. Entrez un nom descriptif pour l'item dans le champ de saisie du titre. Cette entrée est requise aussi.
  4. Dans le champ de saisie de description, vous pouvez entrer les instructions pour les participants du site sur comment sélectionner un item. Ces instructions s'affichent aux participants du site quand ils utilisent le modèle de création de portfolio.
  5. Sous "Autoriser sélection multiple", choisissez une de ces options:
    • Si les créateurs de ces portfolios ont la possibilité d'inclure plus d'une instance de cette information (comme dans le cas de réussites personnelles, expériences professionelles, ou tout autre chose qui peut arriver plus d'une fois), cliquez sur "Oui".
    • Sinon, cliquez sur "Non".
  6. Cliquez sur "Ajout à la liste". Sakai affiche les informations sur cette item en bas de la page.
  7. Pour ajouter un autre item au modèle, répétez les instructions 1 à 6.
  8. Pour changer l'ordre d'affichage des items pour le créateur de portfolio, changez les nombre dans les champs de saisie de la séquence. (Ce changement n'affectera pas la séquence des items dans le portfolio : l'ordre des items dans le portfolio est déterminé par le fichier .xsl que vous avez sélectionné.)
  9. Pour modifier un item que vous avez ajouté, cliquez sur le lien "Editer" en dessous de son nom et effectuez les changements nécessaires. Pour plus d'informations, référez vous aux instructions 1 à 6.
  10. Pour effacer un item que vous avez ajouté, cliquez sur le lien "Effacer" en dessous de son nom. Sakai efface l'item immédiatement. Attention ! Soyez sûr de vouloir effacer l'item avant de cliquer sur "Effacer" car Sakai n'affiche pas de fenêtre de vérification avant d'effacer l'item.
  11. Après avoir ajouté autant d'items que vous le souhaitez, choisissez une des options suivantes :
    • Pour continuer à créer le modèle, cliquez sur "Continuer". Sakai affiche la page de sélection des fichiers de support (étape 4 sur 4).
    • Pour arrêter de travailler sur ce modèle et sauvegarder votre travail, cliquez sur "Finir". Sakai affiche la page d'accueil des modèles portfolio. Votre nouveau modèle est listé sur cette page. Vous avez la possibilité de reprendre votre travail plus tard.
    • Pour revenir à la page de spécification de disposition du modèle (étape 2 sur 4), cliquez sur "Retour".
    • Pour quitter votre travail sans sauvegarder aucune des informations de cette session de création de modèle, cliquez sur "Annuler". Sakai affiche la page d'accueil des modèles portfolio.

Etape 4 : Sélection des fichiers de support

Si vous avez choisi de continuer, Sakai affiche la page de sélection des fichiers de support (étape 4 sur 4) :

Cette page vous permet d'identifier n'importe quel fichier additionel utilisé dans ce modèle. Ces fichiers peuvent inclure des styles de page .css, des images, du son, de la vidéo, du javascript, et tout autre fichier qui détermine le design, l'organisation et tout autre possibilité du modèle de portfolio.

Pour sélectionner ce matériel, suivez ces étapes :

  1. Dans le champ de saisie du nom (utilisé dans xpath), entrez le nom pour ce fichier utilisé dans le fichier .xsl choisi dans l'étape 2 : Spécification de la disposition". Si le modèle est basé sur passthrough.xsl, entrez n'importe quel nom d'un seul mot par lequel vous voulez que le fichier .xsl final identifie ce fichier. Une entrée dans ce champ de saisie est requise.
  2. A la droite de l'espace de choix de fichier, cliquez sur le lien "Sélection d'un fichier" (un choix dans cet espace est requis). Sakai affiche l'écran de sélection de fichier.
  3. Sélectionnez le fichier dont vous avez besoin. Pour ce faire, suivez les instructions pour sélectionner un item. Après avoir cliqué sur "Finir", Sakai affiche la page de sélection des fichiers de support (étape 4 sur 4) à nouveau.
  4. Cliquez sur "Ajouter à la liste". Sakai affiche le nom du fichier en bas de la page.
  5. Pour ajouter un autre fichier de support au modèle, répétez les instructions 1 à 4.
  6. Pour modifier un fichier que vous avez ajouté, cliquez sur le lien "Editer" en dessous de son nom et effectuez les changements nécessaires. Pour plus d'information, voir les instructions 1 à 4.
  7. Pour effacer un fichier que vous avez ajouté, cliquez sur le lien "Effacer" en dessous de son nom. Sakai efface l'item immédiatement. Attention ! Soyez sûr de vouloir effacer l'item avant de cliquer sur "Effacer" car Sakai n'affiche pas de fenêtre de vérification avant d'effacer l'item.
  8. Choisissez une des options suivantes :
    • Pour sauvegarder votre travail, cliquez sur "Finir". Sakai affiche la page d'accueil des modèles portfolio. Ce modèle est listé à l'écran. Vous pourrez travailler dessus votre modèle n'importe quand.
    • Pour retourner à l'écran de listing du contenu (ajout des formes requises et autres items), étape 3 sur 4, cliquez sur "Retour".
    • Pour quitter votre travail sans sauvegarder aucune des informations de cette session de création de modèle, cliquez sur "Annuler". Sakai affiche la page d'accueil des modèles portfolio.

Etape 5 : Achèvement du modèle lorsque passthrough.xsl est utilisé

Si vous avez basé votre modèle sur passthrough.xsl, suivez ces dernières instructions :

  1. Utilisez l'outil de portfolios pour créer un portfolio basé sur ce modèle.
  2. Affichez le portfolio.
  3. Affichez le fichier XML que le système utilise pour générer le fichier HTML final en faisant un clic-droit n'importe où sur le portfolio puis en cliquant sur "code source de la page" dans le menu contextuel.
  4. Utilisez n'importe quel programme créateur de XSLT pour créer un fichier .xsl de transformation.
  5. Utilisez l'outil de modèle de portfolio et remplacez passthrough.xsl par le fichier .xsl que vous avez créé.
