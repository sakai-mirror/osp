Publication d'un modèle

Si vous êtes un administrateur système, vous devez savoir que n'importe quel modèle que vous créez et publiez sur le site PortfolioAdmin devient automatiquement disponible pour tous les sites Sakai sous votre instance.

Après que vous ayez créé un modèle, vous pouvez l'utiliser pour créer votre propre portfolio. Cependant, le modèle n'est pas encore disponible pour l'usage des autres. Pour rendre disponible et utilisable le modèle aux autres participants, vous devez le publier.

Pour ce faire, accédez à l'outil des modèles portfolio et suivez ces étapes :

  1. Si la page d'accueil des modèles portfolio n'est pas affichée, cliquez sur le bouton "Réinitialiser".
  2. Trouvez le modèle que vous souhaitez publier et clichez sur le lien "Publication" en dessous de son nom. Sakai publie le modèle.
