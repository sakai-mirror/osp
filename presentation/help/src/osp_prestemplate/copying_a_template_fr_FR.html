Copie d'un modèle

Pour créer un modèle similaire à un modèle existant, vous pouvez copier un modèle existant et effectuer quelques changements sur la copie. Pour copier un modèle , accédez à l'outil des modèles portfolio et suivez ces étapes :

  1. Si la page d'accueil des modèles portfolio n'est pas affichée, cliquez sur le bouton "Réinitialiser".
  2. Trouvez le modèle que vous voulez utiliser et cliquez sur le lien de copie en dessous du nom du fichier. Sakai copie le modèle et affiche son nom dans la liste.

Le nom du nouveau modèle est le même que celui de l'original, avec le mot "copie" ajouté à la fin.
