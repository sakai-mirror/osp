Modification d'un modèle

Vous pouvez changer le titre et la description pour un article. Vous pouvez aussi rattacher différents fichiers. Pour changer un article, accédez à l'outil de modèles de portfolio et suivez ces étapes :

  1. Si la page de modèles de portfolio n'est pas affichée, cliquez sur le bouton "Réinitialiser".
  2. Cliquez sur le lien de modification en dessous du nom du modèle que vous voulez changer. Sakai affiche le modèle de révision de portfolio, étapes 1 à 4.
  3. Procédez page par page, en modifiant les entrées selon vos souhaits. Pour plus d'information et d'instructions, se référer à l'article "création d'un modèle".
