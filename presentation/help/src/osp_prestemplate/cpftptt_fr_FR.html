Modification des permissions

Pour changer les permissions des utilisateurs dans un ou plusieurs rôles, accédez à l'outil de modèles portfolio et suivez ces étapes :

  1. Si la barre de modèles portfolio n'est pas affichée, cliquez sur le bouton "Réinitialiser".
  2. Cliquez sur le bouton des permissions de la barre de menu. Sakai affiche la page des permissions.
  3. En vous reportant au tableau ci-dessous, cochez les options pour sélectionner les permissions appropriées pour chacun des rôles des utilisateurs.



    Permission Description
    Copier Les utilisateurs dans ce rôle peuvent créer de nouveaux modèles en copiant des modèles existant.
    Publier Les utilisateurs dans ce rôle peuvent publier des modèles pour l'usage des participants du site.
    Effacer Les utilisateurs dans ce rôle peuvent effacer des modèles.
    Créer Les utilisateurs dans ce rôle peuvent utiliser l'outil de modèle portfolio pour créer et importer des modèles.
    Editer Les utilisateurs dans ce rôle peuvent changer les modèles.
    Exporter Les utilisateurs dans ce rôle peuvent télécharger un modèle en fichier compressé sur leur ordinateur.


  4. Pour sauvegarder les réglages de permissions, cliquez sur le bouton "Sauvegarder". Sakai sauvegarde les réglages et affiche la page d'accueil des modèles portfolio. Sinon, pour revenir à la page d'accueil sans sauvegarder vos changements, cliquez sur "Annuler".
