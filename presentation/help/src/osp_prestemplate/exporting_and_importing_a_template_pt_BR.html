Exportar e Importar um Modelo

Se você possui mais de um site, talvez você queira usar o mesmo modelo em dois ou mais deles. Para fazer isso, primeiro exporta o modelo para a sua área de trabalho como um arquivo compactado e depois o importa para outro site onde você pretende utilizá-lo.

Exportar um Modelo

Para exportar um modelo como arquivo compactado, acesse a ferramenta Modelos de Portfólio no site apropriado e siga esses passos:

  1. Se a página inicial de Modelos de Portfólio não for exibida, clique no botão Reset.
  2. Encontre o modelo que você deseja trabalhar e clique no link Exportar abaixo do nome. O Sakai exibirá a janela Abrir Arquivo.
  3. Siga as instruções

Importar um Modelo

Para importar um arquivo compacto do modelo da sua área de trabalho para um site do Sakai, acesse o site que você deseja importar e siga esses passos:

  1. Adicione o arquivo em Repositório.
  2. Acesse a ferramenta Modelos de Portfólio. Se a página inicial não for exibida, clique no botão Reset.
  3. Clique no link Importar na barra de menu de Modelos de Portfólio perto do topo da tela. O Sakai exibirá a tela Importar Modelo.
  4. Clique no link Selecionar Arquivo. O Sakai exibirá Selecionar Item.
  5. Selecione o arquivo compactado para o modelo e siga as instruções para selecionar um item. Depois de clicar em Concluir, o Sakai exibirá a tela Importar Modelo novamente. O nome do arquivo que você escolheu é mostrado na caixa Arquivo do Modelo
  6. Para importar o arquivo, clique em Importar Modelo. O Sakai importará o arquivo e exibirá a página inicial de Modelos de Portfólio. O modelo importado é listado aqui. Ou, para returnar a página inicial sem importar o arquivo, clique em Cancelar.
