Modelos de Portfólio: Resumo

Um modelo de portfólio define o layout, o estilo e o conteúdo de todos os portfólios que são baseados nele. Se você é um organizador site, você pode criar modelos que suportam a finalidade pretendida dos portfólios em seu site. Você pode criar um modelo para suportar currículos ou outros formulários de autoapresentação. Você pode criar um outro para mostrar os resultados do ensino e aprendizagem, como demonstrado nos conteúdos da educação em geral ou matrizes de progresso de cooperação curricular. Ou você pode criar um modelo que é adaptado para avaliação e credenciamento de portfólios, como os que integram o aprendizado ao longo de muitas estruturas de dados e sites para mostrar a aprendizagem em relação aos valores institucionais ou resultados desejados.

Os participantes do site não são obrigados a usar modelos em seus portfólios. Dentro de certos limites, podem também criar seu próprio portfólios.

Modelos de portfólio são mais utilizados quando você, o organizador do site, deseja orientar os participantes do site de maneiras específicas em vez de permitir criar portfólios de acordo com as suas próprias ideias e especificações. A maneira mais eficaz de trabalhar com portfólios e modelos de portfólio é usar modelos para orientar e instruir os participantes na criação de portfólios eficientes, em seguida, fornecer uma variedade de layouts de portfólio e estilos que permite aos participantes em colocar sua própria ideia em portfólios para fins semelhantes ou diferentes.

Cada modelo é baseado em um arquivo .xsl. Este arquivo especifica o layout para o portfólio, os caminhos para o conteúdo do portfólio e, opcionalmente, o caminho para um arquivo .css que define o estilo do portfólio.

Opcionalmente, um modelo também pode incluir "opções de esboço". Estas opções permitem aos participantes adicionar conteúdo aos seus portfólios digitando ou copiando e colando o texto diretamente para ele. Opções de esboço são usados ​​frequentemente quando é mais fácil ou mais intuitivo para os participantes para adicionar o conteúdo diretamente em vez de criar cada item distinto como um formulário nos Recursos. Opções de esboço pode ser usado para solicitar e coletar informações como o nome de um participante do site, data, título e subtítulo para o portfólio, legendas para as imagens e/ ou uma ou mais reflexões. Eles também são utilizados quando o participante precisa especificar um nível específico de uma matriz para incluir em um portfólio.

Opções de esboço são apresentadas ao participante em uma tela do Sakai. Você, o organizador do site, definirá os campos na tela (o que é, essencialmente, um formulário) pela adição de um arquivo .xsd para o modelo.

A ferramenta Portfólio Layouts geralmente está localizada na categoria Design & Publicação de um site portfólio.

Para acessar a ferramenta, clique no botão Iniciar para Modelos de Portfólio. O Sakai exibirá a páina inicial de Modelos de Portfólio.

A tela contem uma lista dos modelos de portfólio de seu site. Se a lista for maior que uma página, use os butões abaixo da barra de menu do Modelos de Portfólio para exibir as outras páginas da lista.

  • Para exibir a primeira página da lista, clique em Primeiro.
  • Para exibir a página anterior da lista, clique em Anterior.
  • Para exibir a página seguinte da lista, clique em Avançar.
  • Para exibir a última página da lista, clique em Última.
