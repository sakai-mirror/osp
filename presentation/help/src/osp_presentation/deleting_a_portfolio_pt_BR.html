Deletar um Portfolio

Você pode deletar qualquer portfolio que criou. Para deletar um portfolio, acesse a ferramenta Portfolios e siga esses passos:

  1. Se a página inicial de Portfólios não for exibida, clique no botão Reset.
  2. Encontre o portfolio que você deseja deletar e clique no link Excluir abaixo do seu nome. O Sakai exibirá a janela Confirmar Exclusão.
  3. Para deletar esse portfolio, clique em OK. O Sakai deleta o portfolio e exibe a página inicial de Portfolios. Ou, para retornar a página sem deletar o portfolio, clique em Cancelar.
