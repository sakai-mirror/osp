Modification d'un portfolio

Vous pouvez modifier tous les portfolios que vous avez créés. Pour modifier un portfolio, allez dans l'outil Portfolios et suivez les étapes suivantes :

  1. Si la page d'accueil des Portfolios n'est pas affichée, appuyez sur le bouton Réinitialiser.
  2. Choisissez le portfolio que vous souhaitez modifier et cliquez sur le lien Modifier sous son nom. Sakai affiche alors les pages de modification de Portfolio, en commençant par l'étape 1 sur 3 : « Début ».
  3. Effectuer les changements voulus dans les champs. Pour plus d'information, voir "Création d'un portfolio", étapes 6 à 10. Poursuivez, selon ce que vous désirez, avec "Conception de votre propre portfolio" ou "Utilisation d'un Modèle pour créer un Portfolio".
