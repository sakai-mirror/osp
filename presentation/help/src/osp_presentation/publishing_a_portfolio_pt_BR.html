Publicar um Portfolio

Depois que você criou um portfolio, você pode publicá-lo para outros visualizarem. Você tem a opção de publicar para um grupo de pessoas do seu site, indivíduos fora de seu site ou fora do ambiente Sakai ou para uma URL que pode ser acessada por alguém através de um navegador web.

Para publicar um portfolio, acesse a ferramenta Portfólio e siga esses passos:

  1. Se a página inicial de Portfólios não for exibida, clique no botão Reset.
  2. Encontre a ferramenta portfolio que você deseja publicar e clique no link Publicar abaixo do seu nome. O Sakai exibirá Selecionar Audiência. Essa tela contem duas seções: Publicar para um Grupo ou Seção Individual e Publicar para seção Web.
    • A seta para a direita no cabeçalho para qualquer seção indica que a seção está fechada. Para abri-la para que você possa ver as caixas nessa seção, clique na seta.
    • Uma seta para baixo no cabeçalho para qualquer seção indica que a seção está aberta (ou seja, suas caixas são exibidas). Para fechá-la, clique na seta.
  3. Para tornar este portfólio disponível para todos os membros do site que estão atribuídos a um papel específico e/ou torná-lo disponível para membros específicos do site, siga estes passos:
    • Em Publicar para de um Grupo ou Seção Individual, clique no papel apropriado ou nome na caixa do papel. (Os papéis são listados em primeiro lugar, seguido por nomes. Você pode precisar deslocar as informações para baixo para ver todos os nomes.) Ou mantenha a tecla CTRL pressionada enquanto clica em múltiplos papéis e/ou nomes.
    • Clique no botão Adicionar. O Sakai exibirá os papeis e os nomes que você escolheu na caixa Audiência Selecionada.
  4. Para compartilhar seu portfólio com alguém que não é um membro do site, digite o endereço de e-mail da pessoa na caixa de e-mail individual e clique em Adicionar. O Sakai acrescenta o endereço para a caixa de Audiência Selecionada. Repita este passo para cada endereço de e-mail que você deseja adicionar.
  5. Para postar seu portfólio na Web, marque a caixa URL Pública em Publicar Seção da Web. O Sakai, em seguida, exibe a URL padrão para os portfólios deste site na caixa à direita. Para especificar uma outra URL, altere a entrada na caixa. Atenção Verifique se a caixa da URL Pública faz seu portfólio publicamente acessível por qualquer pessoa com acesso à web.
  6. Depois de identificar o público para esse portfólio, escolha uma destas opções:
    • Para publicar o portfolio e notificar seu público que já está pronto para visualização, clique em Concluir e Notificar. O Sakai exibirá a tela:
    • Para publicar o portfolio sem notificar o público, clique em Concluir. O Sakai publica o trabalho e exibe a página inicial de Portfolios.
    • Para retornar ao Passo 2 de 3: tela Design, clique em Voltar.
    • Para sair sem publicar o seu portfólio, clique em Cancelar. O Sakai exibirá a página inicial de Portfólios.
  7. Se você clicou em Concluir e Notificar, preencha os campos da tela Notificar Visualizadores como segue:
    • Marque a caixa para cada papel ou usuário individual que você deseja notificar.
    • Em Sua Mensagem, digite uma mensagem que deve ser incluída no e-mail.
  8. Para enviar esta mensagem para os usuários selecionados, clique em Enviar. O Sakai envia a mensagem e exibe a página inicial Portfólios novamente. Ou, para voltar à página inicial sem publicar o portfólio e enviar a mensagem, clique em Cancelar.

Importante Depois de clicar em Enviar, as pessoas que não são membros deste site, mas que estão em seu público selecionado receberá dois e-mails. O primeiro e-mail notificando-os que uma conta foi criada para eles no site e fornece o login e senha necessários para acessá-lo. O segundo e-mail contém informações necessárias para acessar seu portfólio no site. Para visualizar seu portfólio, esses usuários devem inserir as informações de ambos os e-mails. Os membros do site que estão em seu público receberá apenas um e-mail. Este e-mail notifica que seu portfólio está pronto para visualizar e fornece as informações necessárias para acessar o seu portfólio.
