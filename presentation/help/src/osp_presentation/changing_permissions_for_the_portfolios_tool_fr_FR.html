Modification des permissions

Pour modifier les permissions par défaut de l'outil Portfolios, accédez à l'outil et suivez les étapes suivantes :

  1. Si la barre de menu des Portfolios n'est pas affichée, cliquez sur le bouton Réinitialiser.
  2. Cliquez sur le bouton Permissions dans la barre de menu. Sakai affiche l'écran des Permissions.
  3. En vous reportant au tableau ci-dessous, cochez les cases pour attribuer les permissions souhaitées aux utilisateurs de chaque rôle.



    Permission Description
    Supprimer Les utilisateurs de ce rôle peuvent supprimer n'importe quel portfolio qui a été partagé avec eux. (Supprimer une permission ne supprime pas le portfolio d'un autre utilisateur. Les utilisateurs peuvent toujours supprimer leurs propres portfolios, quel que soit le réglage de cette permission.)
    Commenter Les utilisateurs de ce rôle peuvent commenter les portfolios.
    Créer Les utilisateurs de ce rôle peuvent créer leurs propres portfolios.


  4. Pour sauvegarder vos paramètres de permissions, cliquez sur Enregistrer. Sakai enregistre vos paramètres de permissions et affiche la page d'accueil des Portfolios. Sinon, pour retourner à cette page sans enregistrer vos modifications, cliquez sur Annuler.
