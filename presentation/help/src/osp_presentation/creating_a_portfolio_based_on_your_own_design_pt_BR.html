Projetar seu próprio Portfólio

Depois de ter concluído as etapas "Criar um portfólio", acima, o Sakai exibe a Etapa 2 de 3: Tela de Design.

Você pode voltar para a tela Etapa 1 a partir desta tela a qualquer momento. Para fazer isso, clique no botão Voltar perto da parte inferior da tela.

Para continuar, siga estes passos:

  1. Comece a criar uma página no portfólio clicando no link Adicionar Página. Sakai exibe a tela Adicionar Página. Usando esta tela, você especifica o título, a aparência e o conteúdo de uma página de portfólio. A tela contém duas seções: informações sobre a página e o conteúdo da página.
    • A seta para a direita no cabeçalho para qualquer seção indica que a seção está fechada. Para abri-la para que você possa ver as caixas nessa seção, clique na seta.
    • Uma seta para baixo-facing no cabeçalho para qualquer seção indica que a seção está aberta (ou seja, suas caixas são exibidas). Para fechá-la, clique na seta.
  3. Na seção Informações da Página, digite um título para a página de portfólio na caixa Título. A entrada do título é obrigatória.
  4. Se desejar, digite uma descrição da página na caixa Descrição e insera palavras-chave na caixa Palavras-chave.
  5. Clique no link Selecionar Layout. Sakai exibe uma lista de layouts.
  6. Selecione um layout, seguindo as instruções. Depois de ter selecionado um layout, Sakai exibe a tela Adicionar Página novamente. O nome do layout que você escolheu é exibido na caixa Layout, e uma prévia do layout é exibida abaixo da caixa.
  7. Para usar o estilo padrão de sua instituição para fontes, cores e gráficos, deixe em branco a caixa Estilo. Para selecionar um outro estilo para este portfólio, clique no link Selecionar Estilo. Sakai exibe uma lista de estilos para você escolher.
  8. Para selecionar um estilo para este portfolio, siga as instruções. Depois de ter feito a sua seleção, Sakai exibe a tela Adicionar Página novamente. O nome do estilo escolhido é exibido na caixa Estilo. Esse estilo se aplica a todo o portfólio, a menos que você selecione um outro estilo para uma ou mais páginas específicas.
  9. Se a seção Conteúdo da Página estiver fechada, clique na seta no cabeçalho da seção para abri-la. Sakai exibe caixas para o layout que você escolheu.
  10. Digite o texto para cada parte do layout através do editor de texto. Use os ícones e as listas suspensa para formatar o texto conforme o caso. Para adicionar blocos de texto em sequência, clique no botão Adicionar Linha. Sakai exibe uma caixa. Escolha entre as seguintes opções:
    • Para adicionar uma parte do texto, insira-o na caixa. Você pode copiar e colar diretamente no caixa. Apesar de cada caixa ser pequena, você pode digitar o texto, tanto quanto você precisar.
    • Para remover uma caixa de texto, clique no botão Remover Linha.
    • Para adicionar a próxima parte do texto, clique em Adicionar Linha novamente.
  11. Para adicionar esta página ao seu portfólio, clique em Adicionar Página. Sakai exibe a Etapa 2 de 3: Tela de Design novamente. A tela agora mostra um ícone para a página que você adicionou.
  12. Para adicionar outra página, repita os passos 1 - 10.
  13. Para incluir links que permitam aos usuários mover de página a página dentro do portfólio, marque a caixa de Navegação da Página.
  14. Após terminar seu trabalho, escolha uma destas opções:
    • Para salvar seu trabalho e sair, clique em Concluir. Sakai exibe a página inicial Portfólios, e o nome deste portfólio é exibido na lista de portfolio. Você pode retornar a qualquer momento a trabalhar nele.
    • Para publicar este portfólio, clique em Continuar. Sakai exibe a Etapa 3 de 3: Publicar. Os campos são idênticos aos da tela Selecionar Público. Proceda conforme as instruções em "Publicar um Portfólio" início da etapa 3.
    • Para sair sem salvar seu trabalho nesse portfólio, clique em Cancelar. O Sakai exibirá a página inicial de Portfólios.
