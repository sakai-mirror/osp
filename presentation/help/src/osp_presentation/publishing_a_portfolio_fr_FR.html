Publication d'un portfolio

Après avoir créé un portfolio, vous pouvez le publier afin de le rendre visible aux autres. Vous avez la possibilité de le publier aux groupes de personnes sur votre site, à des personnes individuelles hors du site ou hors de l'environnement Sakai ou encore par une adresse accesible par n'importe qui via un navigateur Internet.

Pour publier un portfolio, accédez à l'outil de Portfolio et suivez les étapes suivantes :

  1. Si la page d'accueil des Portfolios n'est pas affichée, cliquez sur le bouton Réinitialiser.
  2. Trouvez le portfolio que vous souhaitez publier et cliquez sur le lien Publier sous son nom. Sakai affiche l'écran Choisir un public. Cet écran comporte deux sections : Publier à un Groupe ou à une Personne et Publier sur le Web.
    • Une flèche vers la droite dans l'en-tête d'une section indique que cette section est fermée. Pour l'ouvrir afin de voir les champs de celle-ci, cliquez sur la flèche.
    • Une flèche vers le bas dans l'en-tête d'une section indique que cette section est ouverte (et donc, que ses champs sont affichés). Pour la fermer, cliquez sur la flèche.
  3. Pour rendre ce portfolio accessible à tous les membres du site assignés à un rôle particulier et/ou pour le rendre accessible à des membres spécifiques du site, suivez ces étapes :
    • Dans la section Publier à un Groupe ou à une Personne, cliquez sur le rôle ou la personne voulue dans le champ Rôle/Nom. (Les rôles sont listés en premier, suivis des noms. Vous risquez de devoir faire défiler vers le bas pour voir tous les noms.) Vous pouvez aussi utiliser la touche Contrôle ou Ctrl pour sélectionner plusieurs rôles et/ou noms.
    • Cliquez sur le bouton Ajouter. Sakai affiche les rôles et noims que vous avez choisis dans le champ Public choisi.
  4. Pour partager votre portfolio avec quelqu'un qui n'est pas membre du site, entrez son adresse email dans le Champ Adresse email individuelle et cliquez sur Ajouter. Sakai ajoute cette adresse au champ Public choisi. Répétez cette étape pour chaque adresse email que vous souhaitez ajouter.
  5. Pour publier votre portfolio sur Internet, cochez la case Adresse publique dans la section Publier sur Internet. Sakai affiche alors l'adresse par défaut des portfolio de ce site dans le champ sur la droite. Pour spécifier une autre adresse, modifier l'information de ce champ. Attention : Cocher la case Adresse publique rend votre portfolio accessible publiquement à n'importe qui ayant accès à Internet.
  6. Après avoir identifié le public de ce portfolio, choisissez une de ces options :
    • Pour publier le portfolio et notifier le public que le site est prêt à être consulté, cliquez sur Terminer et Notifier. Sakai affiche cet écran :
    • Pour publier ce portfolio sans notifier le public, cliquez sur Terminer. Sakai publie votre travail et affiche la page d'accueil des Portfolios.
    • Pour revenir à l'étape 2 sur 3 : Conception, cliquez sur Retour
    • Pour retourner à la page d'accueil sans publier le portfolio, cliquez sur Annuler.
  7. Si vous avez cliqué sur Terminer et Notifier, remplissez les champs de l'écran Destinataires de la Notification comme suit :
    • Cochez la case pour chaque rôle ou utilisateur individuel que vous souhaitez notifier.
    • Dans Votre Message, entrez un message qui sera inclus dans l'email.
  8. Pour envoyer ce message aux destinataires sélectionnés, cliquez sur Envoyer. Sakai envoie le message et affiche à nouveau la page d'accueil des Portfolios. Sinon, pour retourner à la page d'accueil sans publier le portfolio et envoyer de message, cliquez sur Annuler.

Important : Après avoir cliqué sur Envoyer, les personnes qui ne sont pas membres du site mais qui font partie du public sélectionné reçoivent deux emails. Le premier les informe qu'un compte a été créé pour eux sur le site et leur donne un identifiant et un mot de passe nécessaires pour y accéder. Le second email contient les informations nécessaires à l'accès de votre porftolio sur le site. Pour voir votre portfolio, ces utilisateurs devront entrer les informations contenues dans les deux emails. Les membres du site qui ont accès à votre portfolio reçoivent un seul email qui leur notifie que votre portfolio est accessible et leur donne les informations pour y accéder.
