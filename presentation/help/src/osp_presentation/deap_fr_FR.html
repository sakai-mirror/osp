Téléchargement (Export) d'un portfolio

Vous pouvez télécharger (c'est-à-dire, exporter) vers votre poste de travail tout portfolio que vous avez créé. Le portfolio sera exporté dans un fichier compressé (zip) constitué de nombreux fichiers. Pour télécharger un portfolio, accédez à l'outil de Portfolios et suivez les étapes suivantes :

  1. Si la page d'accueil des Portfolios n'est pas affichée, cliquez sur le bouton Réinitialiser.
  2. Trouvez le portfolio que vous souhaitez télécharger et cliquez sur le lien Télécharger sous son nom.
  3. Poursuivre tel qu'indiqué.
