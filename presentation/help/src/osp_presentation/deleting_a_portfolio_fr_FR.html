Suppression d'un portfolio

Vous pouvez supprimer tout portfolio que vous avez créé. Pour supprimer un portfolio, accédez à l'outil Portfolios et suivez les étapes suivantes :

  1. Si la page d'accueil des Portfolios n'est pas affichée, cliquez sur le bouton Réinitialiser.
  2. Trouvez le portfolio que vous souhaitez supprimer et cliquez sur le lien Supprimer sous son nom. Sakai affiche une fenêtre de confirmation de la suppression.
  3. Pour supprimer ce portfolio, cliquez sur OK. Sakai supprime le portfolio et affiche la page d'accueil des Portfolios. Sinon, pour revenir à cette page sans supprimer le portfolio, cliquez sur Annuler.
