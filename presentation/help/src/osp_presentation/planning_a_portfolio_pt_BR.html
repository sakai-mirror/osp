Planejar um Portfólio

Antes de criar um portfólio, recomendamos que você considere cuidadosamente as seguintes perguntas e tome as medidas adequadas.

  • Qual é o objetivo desse portfolio?
  • Qual conteúdo eu preciso incluir, a fim de atingir este objetivo? Quais os arquivos específicos de processamento de texto, planilhas, imagens, e outros eu preciso? Depois de responder a estas perguntas, armazene os arquivos que você pretende usar em Repositório. Para informações sobre como adicionar itens em Repositório
  • Quais informações adicionais (contexto, explicação, e assim por diante) que eu preciso fornecer?
  • Em que ordem eu quero apresentar o material?
  • Qual o controle criativo que eu quero ter sobre a aparência do portfólio? Eu preferiria usar um modelo que controla o layout e texto / cor / estilo de exibição de todo o portfólio ou projetar minha própria abordagem selecionando um conjunto de layouts e estilos predefinidos? Para responder a estas perguntas, recomendamos que você revise todos os modelos apropriados, layouts, e estilos que a sua instituição disponibilizou para o seu uso.
  • Se eu pretendo projetar minha própria abordagem, a apresentação será mais eficaz se eu usar o mesmo layout em cada página da Web ou se as mudanças de layout de página para página? Você pode escolher um layout diferente para cada página ou usar o mesmo layout para todas as páginas na sua apresentação.
  • Com base no layout(s) que eu escolhi, quantas páginas serão necessárias?
  • Se eu pretendo projetar minha própria abordagem, a apresentação será mais eficaz se eu usar o mesmo conjunto de fontes, cores e tamanhos de texto, espaçamento e cores de fundo para todas as páginas? Ou deveria a exibição de texto e do fundo mudar de página em página? Sua instituição tem um "estilo padrão" que inclui um conjunto padrão de fontes, cores, e assim por diante. Se a sua instituição criou outros estilos também, você pode escolher entre eles para cada página ou você pode optar por utilizar um estilo único para toda a apresentação.
