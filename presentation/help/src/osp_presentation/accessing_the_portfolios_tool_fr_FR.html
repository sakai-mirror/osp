Accès à l'outil de portfolios

Cliquez sur l'outil Portfolios pour y accéder. Sakai affiche alors la page d'accueil des portfolios.

Cette page contient une liste de tous les portfolios du site auxquels vous êtes autorisés à accéder. Si cette liste s'étend sur plus d'une page, utilisez les boutons sous la barre de menus Portfolios pour afficher les autres pages de la liste.

  • Pour afficher la première page de la liste, cliquer sur Début.
  • Pour afficher la page précédente de la liste, cliquer sur Précédent.
  • Pour afficher la prochaine page de la liste, cliquer sur Suivant.
  • Pour afficher la dernière page de la liste, cliquer sur Fin.
