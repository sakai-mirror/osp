Criar um Portfólio

O processo de criação de um portfólio envolve a seleção de itens de Repositório, dando ao portfólio um nome e uma data de expiração opcional, decidindo se deve ou não permitir comentários, determinando quais os participantes do site ou usuários fora do site terão acesso ao portfólio, decidindo se deve ou não para tornar o portfólio público, e outros, opcionalmente, notificando que um portfólio foi compartilhado com eles.

Para criar um portfólio, acesse a ferramenta Portfólios e siga estes passos:

  1. Se a página inicial de Portfólios não for exibida, clique no botão Reset.
  2. Na barra de menu de Portfólios perto do topo da tela, clique no botão Adicionar. O Sakai exibe a tela Adicionar Portfolio.
  3. Decida se você deseja criar um portfolio "do zero" ou baseá-lo em um modelo.
    • Para criar um portfólio no qual você escolhe livremente o layout, estilos de texto e conteúdo para incluir, clique no botão Desenvolva seu próprio portfólio.
    • Para basear o seu portfólio em um modelo existente, clique no botão Usar um modelo.
  4. Se você optar por usar um modelo, selecione o modelo que deseja usar na lista suspensa. Se desejar, visualize o modelo que você escolheu. Para fazer isso, clique em Visualizar Modelo. Para fechar a tela de visualização, clique em Fechar visualização.
  5. Para prosseguir, clique em Continuar. Ou para sair sem criar um portfólio clique em Cancelar. Se você optar por continuar, o Sakai exibe a tela Passo 1 de 3: Começar.
  6. Digite um título (obrigatório) e uma descrição deste portfólio nas caixas fornecidas.
  7. Para especificar a última data em que este portfólio está disponível para visualização, use as listas suspensas na caixa de Expira em, insira essa data.
  8. Para permitir que outras pessoas vejam e comentam este portfolio, clique em permitir comentários.
  9. Após terminar seu trabalho, escolha uma destas opções:
    • Para continuar a trabalhar nesse portfólio, clique em Continuar.
    • Para salvar seu trabalho e sair, clique em Concluir. Sakai exibe a página inicial Portfólios, e o nome deste portfólio é exibido na lista de portfólio. Você pode retornar a qualquer momento para continuar a trabalhar nesse portfólio.
    • Para sair sem salvar seu trabalho nesse portfólio, clique em Cancelar. O Sakai exibirá a página inicial de Portfólios.
  10. Se você selecionou para continuar, siga as instruções dadas em "Projetar seu próprio portfolio", ou "Criar um portfólio baseado em modelos".
