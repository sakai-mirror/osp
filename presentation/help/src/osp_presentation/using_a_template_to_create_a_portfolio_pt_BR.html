Criar um Portfólio Baseado no Modelo

Depois que você completou os passos 1 a 10 em "Criar um Portfólio", você deve digitar as informações de uma ou mais telas. Se você vê uma ou duas telas, depende do modelo que você escolheu.

Se foi apresentada duas telas, a primeira tela pode ser parecida com isso:

Normalmente, essa tela irá exigir que você faça um ou ambos destes procedimentos:

  • Fornecer texto para um título do portfolio, um subtítulo e/ou legendas para as imagens.
  • Especifique um determinado nível de uma matriz para incluir no portfólio. A segunda tela contém listas suspensas a partir do qual você seleciona os itens para incluir em seu portfólio.

Complete a(s) tela(s), seguindo estes passos:

  1. Digite as informações e anexe arquivos e outros materiais, conforme as instruções na tela. Em geral:
    • Para digitar o texto em uma caixa,digite-o ou copie e cole.
    • Para anexar um item que você tem armazenado em Repositório, siga as instruções.
  2. Quando você estiver pronto para sair desta tela, escolha uma destas opções:
    • Para salvar seu trabalho e sair, clique em Concluir. O Sakai exibe a página inicial Portfólios, e o nome deste portfólio é exibido na lista de portfolio. Você pode retornar a qualquer momento para continuar a trabalhar nesse portfólio.
    • Para prosseguir, clique em Continuar. O Sakai exibe a segunda de duas telas ou exibe uma tela na qual você pode publicar seu portfólio. Para obter informações sobre a publicação de um portfólio, consulte "Publicar um Portfolio".
    • Para sair sem salvar seu trabalho nesse portfólio, clique em Cancelar. O Sakai exibirá a página inicial de Portfólios.
    • Para voltar à tela anterior, clique no botão Voltar perto da parte inferior da tela.
