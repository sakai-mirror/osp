Conception de votre propre portfolio

Après avoir suivi les étapes de la page "Création d'un Portfolio", Sakai affiche en dessous l'écran de l'étape 2 sur 3 : Conception.

Vous pouvez retourner à l'écran de l'étape 1 depuis cet écran à tout moment. Pour ce faire, cliquez sur le bouton Retour au bas de l'écran.

Pour procéder à la conception de votre portfolio, suivez les étapes suivantes :

  1. Commencez par créer une page dans le portfolio en cliquant sur le lien Ajouter une page. Sakai affiche alors l'écran Ajouter une Page. Sur cet écran, vous spécifiez le titre, l'apparence et le contenu d'une page du portfolio. Cet écran comporte 2 sections : Informations sur la page et Contenu de la page.
    • Une flèche vers la droite dans l'en-tête d'une section indique que cette section est fermée. Pour l'ouvrir afin de voir les champs de celle-ci, cliquez sur la flèche.
    • Une flèche vers le bas dans l'en-tête d'une section indique que cette section est ouverte (et donc, que ses champs sont affichés). Pour la fermer, cliquez sur la flèche.
  3. Dans la section Informations sur la page, entrez un titre pour la page de portfolio dans le champ Titre. Ce champ est obligatoire.
  4. Si vous le souhaitez, entrez une description de la page dans le champ Description et des mots-clés dans le champ Mots-clés.
  5. Cliquez sur le lien Sélectionnez une Mise en page. Sakai vous propose une liste de mises en page.
  6. Sélectionnez une mise en page en suivant les instructions. Une fois que vous avez sélectionné une mise en page, Sakai affiche à nouveau l'écran Ajouter une page. Le nom de la mise en page que vous avez choisie est affiché dans le champ Mise en page, et un aperçu de cette mise en page est affiché sous le champ.
  7. Pour utiliser le style par défaut de votre institution pour les polices, couleurs et images, laissez le champ Style vide. Pour sélectionner un autre style pour ce portfolio, cliquez sur le lien Choisir un style. Sakai affiche alors une liste de styles parmi lesquels effectuer votre choix.
  8. Pour choisir un style pour ce portfolio, suivez les instructions. Une fois votre sélection effectuée, Sakai affiche à nouveau l'écran Ajouter une page. Le nom du style que vous avez choisi est affiché dans le champ Style. Ce style s'applique à tout le portfolio sauf si vous spécifiez un autre style pour une ou plusieurs pages.
  9. Si la section Contenu de la page est fermée, cliquez sur la flèche dans l'en-tête pour l'ouvrir. Sakai affiche les champs correspondants à la mise en page que vous avez choisie.
  10. Entrez du texte pour chaque partie de la mise en page grâce à l'éditeur de texte avancé. Utilisez les icônes et les listes déroulantes pour formater le texte comme vous le souhaitez. Pour ajouter des blocs de texte sous forme de liste, utilisez les boutons adéquats dans la barre de menus de l'éditeur de texte.
  11. Pour ajouter cette page à votre portfolio, cliquez sur Ajouter la page. Sakai affiche à nouveau l'écran de l'étape 2 sur 3 : Conception. L'écran affiche maintenant une icône pour la page que vous avez ajoutée.
  12. Pour ajouter une autre page, répétez les étapes 1 à 10.
  13. Pour insérer des liens qui permettent aux utilisateurs de se déplacer entre vos pages dans le portfolio, consultez le champ Navigation entre les pages.
  14. Quand vous avez terminé votre travail, choisissez une des options suivantes :
    • Pour enregistrer votre travail et quitter, cliquez sur Terminer. Sakai affiche la page d'accueil des Portfolios, et le nom du portfolio est affiché dans la liste des portfolios. Vous pouvez ainsi y revenir à tout moment pour y travailler.
    • Pour publier ce portfolio, cliquez sur Continuer. Sakai affiche l'écran de l'étape 3 sur 3 : Publication. Les champs sont les mêmes que pour l'écran de sélection du public. Procédez comme indiqué dans "Publication d'un portfolio" en commençant par l'étape 3.
    • Pour quitter sans enregistrer votre travail pour ce portfolio, cliquez sur Annuler. Sakai affiche la page d'accueil des Portfolios.
