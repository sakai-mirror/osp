Création d'un portfolio

Le processus de création d'un portfolio implique de sélectionner des items depuis les Ressources, de donner un nom au portfolio, éventuellement une date d'expiration, de décider d'autoriser ou non les commentaires, de déterminer quels participants au site ou utilisateurs extérieurs y ont accès, de décider de rendre le portfolio public ou non, et éventuellement d'informer les autres qu'un portfolio a été partagé avec eux.

Pour créer un portfolio, allez à l'outil Portfolios et suivez les étapes suivantes :

  1. Si la page d'accueil des Portfolios n'est pas affichée, cliquez sur le bouton Réinitialiser.
  2. Dans la barre de menus en haut de l'écran, cliquez sur le bouton Créer un nouveau portfolio. Sakai affiche l'écran Créer un Portfolio.
  3. Choisissez si vous souhaitez créer un portfolio en ne partant de rien, ou bien en vous basant sur un modèle.
    • Pour créer un portfolio dans lequel vous êtes libre de choisir la mise en page, le style des textes, et le contenu à inclure, cliquez sur le bouton Concevez votre propre portfolio.
    • Pour baser votre portfolio sur un modèle existant, cliquez sur le bouton Utiliser un modèle.
  4. Si vous avez choisi d'utiliser un modèle, sélectionnez celui que vous souhaitez utiliser depuis la liste déroulante. Si vous le désirez, prévisualisez le modèle que vous avez choisi. Pour ce faire, cliquez sur Prévisualiser le modèle. Pour fermer l'affichage de l'aperçu, cliquez sur Fermer l'aperçu.
  5. Pour valider, cliquez sur Continuer. Pour quitter sans créer de portfolio, cliquez sur Annuler. Si vous avez choisi de continuer, Sakai affiche l'écran de l'étape 1 sur 3 : Début.
  6. Entrez un titre (obligatoire) et une description de ce portfolio dans les champs prévus à cet effet.
  7. Pour indiquer la date limite jusqu'à laquelle ce portfolio est visible, utilisez les listes déroulantes dans les champs Expiration.
  8. Pour autoriser la vue et la publication de commentaires sur ce portfolio, cliquez sur autoriser les commentaires.
  9. Quand vous avez terminé votre travail, choisissez une des ces options :
    • Pour continuer à travailler sur ce portfolio, cliquez sur Continuer.
    • Pour enregistrer votre travail et quitter, cliquez sur Terminer. Sakai affiche la page d'accueil des Portfolios, et le nom de ce portfolio est affiché dans la liste des portfolios. Vous pouvez ainsi y revenir à tout moment.
    • Pour quitter sans enregistrer votre travail pour ce portfolio, cliquez sur Annuler. Sakai affiche la page d'accueil des Portfolios.
  10. Si vous avez choisi de continuer, suivez les instructions données dans "Conception de votre propre Portfolio" ou "Création d'un Portfolio basé sur un modèle". Ce choix dépend du choix que vous avez effectué dans l'étape 3.
