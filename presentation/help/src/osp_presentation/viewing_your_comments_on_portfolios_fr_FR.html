Consultation de vos commentaires sur les portfolios

Pour lire les commentaires que vous avez déposés sur les portfolios de ce site, accédez à l'outil Portfolios et suivez ces étapes :

  1. Si la page d'accueil des Portfolios n'est pas affichée, cliquez sur le bouton Réinitialiser.
  2. Dans le menu Portfolios en haut de la page, cliquez sur le bouton Mes commentaires. Sakai affiche l'écran des Commentaires. Les informations de chaque commentaire sont affichées sur deux lignes.
    • La première ligne indique le nom du portfolio et le modèle utilisé pour le créer (le cas échéant), le titre du commentaire et la date et l'heure à laquelle il a été déposé, et le nom de l'auteur. La dernière colonne indique si le commentaire est visible pour tout le monde (Public), seulement pour vous et l'auteur du commentaire (Partagé) ou bien seulement pour vous (Privé).
    • La seconde ligne contient le texte du commentaire.
  3. Si vous le souhaitez, vous pouvez trier la liste par n'importe laquelle des options de l'en-tête de colonne. Pour cela, cliquez simplement sur une option pour faire le tri. La première fois que vous cliquez sur une option, Sakai trie la liste dans l'ordre ascendant. Si vous cliquez à nouveau sur l'option, Sakai trie la liste dans l'ordre descendant.
  4. Pour retourner à la page d'accueil des Portfolios, cliquez sur le bouton Réinitialiser.
