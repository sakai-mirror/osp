Création d'un portfolio basé sur un modèle

Une fois que vous avez effectué les étapes 1 à 10 de "Création d'un Portfolio", vous devez encore saisir des informations dans quelques écrans. Selon le modèle que vous avez choisi, vous aurez un ou deux écrans à compléter.

Si vous avez affaire à deux écrans, le premier devrait ressembler à ceci :

  • Mettre du texte dans les champs du titre, du sous-titre et/ou des étiquettes d'images de votre portfolio.
  • Renseigner un certain niveau d'une matrice à inclure dans votre portfolio. Le second écran contient des listes déroulantes depuis lesquelles vous sélectionnez des items à inclure dans votre portfolio.

Complétez le(s) écran(s) en suivant les étapes suivantes :

  1. Entrez des informations et des joignez des fichiers ou toute autre support comme indiqué à l'écran. De manière générale :
    • Pour entrer du texte dans un champ, vous pouvez soit le taper au clavier, soit le copier-coller.
    • Pour joindre un item que vous avez stocké dans les Ressources, suivez les instructions.
  2. Quand vous êtes prêt à quitter cet écran, choisissez une des ces options :
    • Pour enregistrer votre travail et quitter, cliquez sur Terminer. Sakai affiche la page d'accueil des Portfolios, et le nom de ce portfolio est affiché dans la liste des portfolios. Vous pouvez ainsi y revenir à tout moment pour y travailler.
    • Pour poursuivre, cliquez sur Continuer. Sakai affiche alors le second des deux écrans ou affiche un écran depuis lequel vous pouvez publier votre portfolio. Pour des informations sur la publication d'un portfolio, voir "Publication d'un Portfolio".
    • Pour quitter sans enregistrer votre travail pour ce portfolio, cliquez sur Annuler. Sakai affiche la page d'accueil des Portfolios.
    • Pour revenir à l'écran précédent, cliquez sur le bouton Retour en bas de la page.
