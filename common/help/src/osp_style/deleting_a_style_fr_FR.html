Suppression d'un style

Si vous êtes organisateur de site, vous pouvez supprimer un style s'il remplit ces conditions:

  • Il n'a pas encore été publié. Une fois qu'un style a été publié, seul l'administrateur système peut le supprimer.

Pour supprimer un style, accédez à l'outil Styles et suivez ces étapes :

  1. Si la page d'accueil Styles n'est pas affichée, cliquez sur le bouton Réinitialiser.
  2. Choisissez le style à supprimer et cliquez sur le lien Supprimer en dessous de son nom. Sakai affiche une fenêtre de confirmation de suppression.
  3. Pour supprimer le style, cliquez sur OK. Sakai supprime le style et affiche la page d'accueil Styles. Pour revenir à la page d'accueil sans supprimer ce style, cliquez sur Annuler.
