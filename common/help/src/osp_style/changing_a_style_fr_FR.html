Modification d'un style

Si vous êtes organisateur de site, vous pouvez changer le nom et la description de tous les styles que vous avez créés qui n'ont pas été publiés. Vous pouvez également choisir un autre fichier .css pour tout style non-publié. Une fois qu'un style a été publié, seul l'administrateur système peut le modifier.

Pour modifier un style, accédez à l'outil Styles et suivez ces étapes :

  1. Si la page d'accueil Styles n'est pas affichée, cliquez sur le bouton Réinitialiser.
  2. Trouvez le style que vous souhaitez modifier, puis cliquez sur le lien Modifier en dessous de son nom. Sakai affiche l'écran de modification du Style.
  3. Modifier les champs selon les besoins.
  4. Pour enregistrer vos modifications, cliquez sur Enregistrer les modifications. Pour revenir à cette page sans enregistrer vos modifications, cliquez sur Annuler.
