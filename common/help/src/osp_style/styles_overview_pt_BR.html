Estilos: Resumo

Estilos são conjuntos de fontes, cores e gráficos. Os organizadores do site usam estilos para controlar a aparência de células da matriz, assistentes e páginas do assistente. Participantes do site usá-los para controlar a aparência de modelos projetaram.

Diretrizes para Usar Estilos A fim de manter a uniformidade dentro de uma matriz ou assistente, recomendamos que você, o organizador do site, aplique o mesmo estilo para todas as células da matriz ou todas as páginas do assistente. Portfólios não requerem necessariamente essa coerência. Participantes do site que desenham seus próprios portfólios podem usar mais de um estilo. Para esses usuários, é mais útil ter uma variedade de estilos disponíveis para o seu trabalho.

Se você é um organizador de site, você usa a ferramenta Estilos para "criar" um estilo, carregando uma folha de estilo em cascata (ou seja, um arquivo .css) do Repositório, dando-lhe um nome significativo, e (se desejado) fornecendo uma descrição do mesmo. Esta ferramenta também permite que você altere o nome e a descrição de um estilo existente, exclue um estilo que você não precisa mais, e altere as permissões de uso de todos os usuários em qualquer papel.

Uma vez que o estilo foi criado em um Site ou globalmente usando o site Portfólio Admin, o estilo estará disponível para todos os membros do site para usar.

A ferramenta Estilos geralmente está localizada na categoria Design & Publicação de um site Portfolo. Para acessar a ferramenta, clique no botão Iniciar para Estilos. O CLE exibe a página inicial de Estilos.

Esta página contém uma lista de todos os estilos no site que você criou e que foram publicadas para que outros o usem. Para cada estilo na lista, você pode ver uma descrição, o nome da pessoa que criou o estilo, e se tem sido sugerido para a publicação.

Se a lista é maior que uma página, use os botões abaixo da barra de menu de Estilos para exibir outras páginas.

  • Para exibir a primeira página da lista, clique em Primeiro.
  • Para exibir a página anterior da lista, clique em Anterior.
  • Para exibir a página seguinte da lista, clique em Avançar.
  • Para exibir a última página da lista, clique em Última.
