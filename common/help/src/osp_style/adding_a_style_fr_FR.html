Ajout d'un style

Si vous êtes administrateur système, sachez que tous les styles que vous ajoutez à la communauté PortfolioAdmin deviennent automatiquement disponibles pour tous les sites de votre instance.

Pour ajouter un nouveau style à votre site, suivez ces étapes :

  1. Ajoutez le fichier .css de ce style aux ressources.
  2. Accéder à l'outil Styles.
  3. Si la page d'accueil Styles n'est pas affichée, cliquez sur le bouton Réinitialiser.
  4. Dans la barre de menu en haut de l'écran, cliquez sur le bouton Ajouter. Sakai affiche l'écran d'ajout de Style.
  5. Dans la zone Nom, entrez un nom descriptif pour ce style.
  6. Si vous le souhaitez, saisissez une description plus détaillée de ce style dans la zone Description.
  7. À la droite de la zone Fichier css, cliquez sur le lien Sélectionner dossier.
  8. Sélectionnez le fichier .css dont vous avez besoin en suivant les instructions pour sélectionner un élément. Après avoir cliqué sur Terminer, Sakai affiche l'écran Ajouter nouveau style. Le nom du fichier que vous avez sélectionné est affiché dans la boîte.
  9. Pour terminer votre travail, cliquez sur Ajouter Style. Sakai affiche de nouveau la page d'accueil Styles. Le nouveau style a été ajouté à la liste et est disponible pour les participants du site. Pour revenir à la page d'accueil sans ajouter le nouveau style, cliquez sur Annuler.
