Publicar um Estilo Globalmente

Se você é um administrador de sistema, você pode publicar um estilo global (isto é, para todos os sites em uma instância CLE). Para isso, faça o login e siga estes passos:

  1. Clique na pasta Administração no topo da tela.
  2. Selecione o site PortfolioAdmin.
  3. Acesse a ferramenta Estilos. O Sakai exibirá a página inicial de Estilos.
  4. Encontre o estilo que você deseja publicar e clique no link Publicar abaixo de seu nome. O Sakai faz o estilo disponível para todos os sites.
