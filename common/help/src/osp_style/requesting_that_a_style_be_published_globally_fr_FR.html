Requête pour qu'un style soit publié globalement

Si vous avez créé un style qui est destiné à une application générale, vous pouvez souhaiter le rendre disponible pour une utilisation dans tous les sites. Toutefois, seuls les administrateurs système peuvent publier un style au niveau global (c.a.d., pour toutes les communautés de Sakai).

Pour demander qu'un style soit publié globalement, accédez à l'outil Styles et suivez ces étapes:

  1. Si la page d'accueil Styles n'est pas affichée, cliquez sur le bouton Réinitialiser.
  2. Cliquez sur le lien Suggérer pour une publication globale sous le nom du style. Sakai enregistre la demande pour la publication globale et affiche de nouveau la page d'accueil Styles.
