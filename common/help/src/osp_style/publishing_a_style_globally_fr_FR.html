Publication globale d'un style

Si vous êtes un administrateur système, vous pouvez publier un style au niveau global (c'est à dire pour tous les sites d'une instance Sakai). Pour ce faire, connectez-vous et suivez ces étapes:

  1. Cliquez sur le dossier d'administration dans le haut de l'écran.
  2. Sélectionnez la communauté PortfolioAdmin.
  3. Accédez à l'outil Styles. Sakai affiche la page d'accueil Styles.
  4. Trouvez le style que vous souhaitez publier et cliquez sur le lien "Publier sous son nom. Sakai rend le style disponible pour tous les sites.
