Modifier les autorisations

Pour modifier les autorisations pour les utilisateurs dans un ou plusieurs rôles, accéder à l'outil Styles et suivez ces étapes:

  1. Si la barre de menu Styles n'est pas affichée, cliquez sur le bouton Reset.
  2. Cliquez sur le bouton Autorisations sur la barre de menu. Sakai affiche l'écran des autorisations.
  3. Se référant à la table ci-dessous, cochez les cases pour sélectionner les autorisations appropriées pour tous les utilisateurs selon leur rôle.



    Autorisations Déscription
    Global Publish Les utilisateurs ayant ce rôle peuvent publier globalement (c. a. d. pour tous les sites). Seul l'administrateur système peut publier globalement un style.
    Publish Les utilisateurs ayant ce rôle peuvent publier les styles pour une utilisation par les membres du site.
    Delete Les utilisateurs ayant ce rôle peuvent supprimer les styles.
    Create Les utilisateurs ayant ce rôle peuvent créer les styles.
    Edit Les utilisateurs ayant ce rôle peuvent modifier les styles.
    Suggest Global Publish Les utilisateurs ayant ce rôle peuvent suggérer que le style soit publié globalement.


  4. Pour enregistrer les paramètres des autorisations, cliquez sur le bouton Enregistrer. Sakai enregistre les paramètres et affiche la page d'accueil Styles. Ou, pour Retour à la page d'accueil sans enregistrer vos modifications, cliquez sur Annuler.
