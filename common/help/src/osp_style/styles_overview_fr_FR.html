Styles: Généralités

Les styles sont des ensembles de polices, de couleurs et de graphiques. Les administrateurs du site utilisent les styles pour contrôler l'apparence de cellules de la matrice ou des Assistants. Les participants du site les utilisent pour contrôler l'apparence des modèles qu'ils ont conçus.

Conseils d'utilisation Styles Afin de maintenir la cohérence au sein d'une matrice ou d'assistant, nous vous recommandons, à vous, l'organisateur du site, d'appliquer le même style à toutes les cellules de la matrice ou toutes les pages de l'Assistant. Les Portfolios ne nécessitent pas forcément cette cohérence. Les participants qui conçoivent leurs propres portfolios peuvent souhaiter utiliser plus d'un style. Ces utilisateurs trouvent plus utile d'avoir une variété de styles disponibles pour leur travail.

Si vous êtes administrateur de site, vous utilisez l'outil de styles pour "créer" un style en téléchargeant une feuille de style en cascade (c'est-à-dire, un .css) à partir de ressources, lui donnant un nom significatif, et (si besoin) en fournissant une description de celui-ci. Cet outil permet aussi de changer le nom et la description d'un style existant, supprimer un style, si vous n'en avez plus besoin, et de changer les autorisations d'utilisation pour tous les utilisateurs de n'importe quel rôle.

Une fois qu'un style a été créé dans un site ou après avoir utilisé la publication globale d’un style dans la communauté Portfolio Admin, le style est disponible pour tous les membres du site concerné.

L'outil Styles est généralement situé dans la partie conception d'un site Portfolo. Pour accéder à l'outil, cliquez sur le bouton Styles. Sakai affiche la page d'accueil Styles.

Cette page contient une liste de tous les styles sur le site que vous avez créés et qui ont été publiés dans le site. Pour chaque style dans la liste, vous pouvez voir sa description, le nom de la personne qui l'a créé le style, et s'il a été proposé pour la publication globale.

Si la liste ne tient pas sur une page, utilisez les boutons sous la barre de menu Styles pour afficher les autres pages.

  • Pour afficher la première page de la liste, cliquez Premier.
  • Pour afficher la page précédente de la liste, cliquez sur Précédent.
  • Pour afficher la page suivante de la liste, cliquez sur Suivant.
  • Pour afficher la dernière page de la liste, cliquez sur Dernier.
