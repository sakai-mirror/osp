Modification d'un terme du glossaire

Pour modifier un terme de glossaire ou une de ses définitions, accédez à l'outil Glossaire et suivez ces étapes :

  1. Si vous ne voyez pas la barre de menu Glossaire, cliquez sur le bouton Réinitialiser.
  2. Cliquez sur le lien Réviser pour le terme à modifier. Sakai affiche l'écran de modification du terme du Glossaire.
  3. Modifiez les entrées description courte et longue via leurs boîtes de description, si besoin. Utilisez les icônes et les listes déroulantes pour formater le texte pour la description longue, selon le cas.
  4. Cliquez sur le bouton "Enregistrer les modifications". Sakai génère les modifications et affiche la liste sur la page d'accueil. Le terme modifié et sa définition courte sont affichés.

Pour revenir à la page d'accueil sans en changer l'entrée du lexique, cliquez sur Annuler.
