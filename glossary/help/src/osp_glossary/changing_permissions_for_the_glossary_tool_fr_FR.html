Modification des permissions

Tous les participants qui travaillent sur une matrice peuvent afficher les définitions d'un glossaire.

Les possibilités pour ajouter, supprimer, changer ou exporter des termes de glossaire sont définies par les permissions attribuées au sein des rôles, comme d'autres outils dans Sakaï.

Pour attribuer ces permisisons, cliquer sur l'outil Glossaire et suivre ces étapes :

  1. Si vous ne voyez par l'outil Glossaire dans la barre de menu sur la gauche, cliquer sur mettre à jour
  2. Cliquer sur le lien Permissions dans la barre de menu haute. Sakai affiche un "tableau".
  3. En se référant au tableau ci-dessous, cocher les cases pour attribuer les différentes permissions au sein de chaque rôle.



    Permission Description
    Supprimer Un utilisateur possédant cette permission dans un rôle peut supprimer des termes du glossaire
    Ajouter Un utilisateur possédant cette permission dans un rôle peut ajouter et importer des termes dans le glossaire.
    Réviser Un utilisateur possédant cette permission dans un rôle peut changer des termes du glossaire et leurs définitions.
    Exporter Un utilisateur possédant cette permission dans un rôle peut exporter en fichier zip les termes du glossaire.

Pour enregistrer les permissions attribuées, cliquer sur enregistrer. Sakai enregistre les modifications et affiche la page d'accueil du glossaire. Ou pour retourner sur la page d'accueil du glossaire sans enregistrer les modifications, cliquer sur Annuler.
