Ajout d'un terme de glossaire

Pour ajouter un terme au glossaire, accédez à l'outil Glossaire et suivez ces étapes :

  1. Si vous ne voyez pas la barre de menu Glossaire, cliquez sur le bouton Réinitialiser
  2. Cliquez sur le lien Ajouter de la barre de menu. Sakai affiche l'écran Ajouter terme de glossaire.
  3. Dans la zone Terme, entrez le terme que vous devez définir.
  4. Dans la boîte description courte, entrez une brève définition du terme.
  5. Dans la boîte de description longue, entrez une définition, ou une description ou des informations plus complètes pour permettre aux participants de bien comprendre le terme. Utilisez les icônes et les listes déroulantes pour formater le texte selon le cas
  6. Cliquez sur le bouton Ajouter un terme. Sakai, ajoute les mots dans le glossaire et l'affiche dans la liste sur la page d'accueil. Le nouveau terme et sa définition courte sont affichés dans la liste.

Ou, pour revenir à la page d'accueil sans ajouter le terme, cliquez sur Annuler.
