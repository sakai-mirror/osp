Exportation et importation d'un terme du glossaire

Si vous possédez plus d'un site, vous pourriez vouloir utiliser le même glossaire dans deux ou plusieurs d'entre eux. Pour ce faire, vous devez d'abord exporter le glossaire sur votre poste de travail sous forme de fichier compressé et ensuite l'importer sur les sites où vous avez prévu de l'utiliser.

Exporter un glossaire

Pour exporter un glossaire sous forme de fochier compressé, accéder à l'outil Glossaire dans le site approprié et suivez les étapes suivantes

  1. Si vous ne voyez pas la barre de menu du Glossaire, cliquez sur Réinitialiser.
  2. Cliquer sur le lien Exporter dans la barre de menus. Sakai ouvre une fenêtre.
  3. Suivez les instructions.

Importation d'un glossaire

Pour importer un fichier de glossaire compressé, accéder au site sur lequel vous souhaitez l'importer et suivez les étapes suivantes:

  1. Ajoutez le fichier aux Ressources.
  2. Accédez à l'outil Glossaire. Si la barre de menu du Glossaire n'est pas affichée, cliquez sur le bouton Réinitialiser.
  3. Cliquez sur le bouton Importer de la barre de le menu. Sakai affiche l'écran Importer des termes du glossaire.
  4. Cliquez sur Choisir un fichier. Sakai affiche l'écran Choisir un item.
  5. Choisissez le fichier compressé du glossaire.
  6. Choisissez si vous souhaitez qu'un terme qui existe soit remplacé par le terme d'importation ou si vous préférez qu'il soit ignoré en cliquant sur le bouton radio approprié.
  7. Pour importer le fichier, cliquez sur "Importer". Sakai importe le fichier, ajoute les les termes au Glossaire, et affiche l'écran d'accueil du Glossaire. Ou pour revenir à la page d'accueil sans importer le fichier, cliquez sur "Annuler".
