Publicar um Termo do Glossário Globalmente

Se você é um administrador de sistema, você pode publicar um termo do glossário globalmente (isto é, para todos os sites em uma instância do Sakai). Para isso, faça o login e siga estes passos:

  1. Clique na pasta Administração no topo da tela.
  2. Selecione o site PortfolioAdmin.
  3. Acesse a ferramenta Glossário. O Sakai exibirá a página inicial do Glossário.
  4. Adicione o termo conforme as instruções acima. Por você estar trabalhando no site PortfolioAdmin, o Sakai tornará este termo disponível para todos os sites.
