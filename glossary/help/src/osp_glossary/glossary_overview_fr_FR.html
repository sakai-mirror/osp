Glossaire : Généralités

L'outil Glossaire fournit un complément d'information sur les lignes et les colonnes des matrices.

Quand les participants déplacent la souris sur le nom d'une ligne ou d'une colonne dans une matrice, Sakai affiche une brève définition ou explication qui a été saisie grâce à l'outil Glossaire. Lorsque les participants cliquent sur le nom, il s'établit un hyperlien vers des informations complémentaires prévues par l'organisateur site. Les organisateurs du site ont souvent recours à ces hyperliens pour fournir des informations détaillées sur les objectifs, les résultats, ou les normes représentées par chaque critère, une liste complète des activités sur le campus pour chaque catégorie, ou un groupement étendu de ressources pour développer chacune des compétences.

Si vous êtes un administrateur système ou un instructeur, l'outil glossaire vous permet de créer des définitions courtes et longues et les explications décrites ci-dessus. Si vous êtes un administrateur système, vous pouvez également déclarer les termes du glossaire pour une utilisation globale (sur l'ensemble des communautés). Dans ce cas, les termes seront affichés dans tous les sites qui utilisent l'outil Glossaire.

Cet outil vous permet également de gérer les termes du glossaire. Vous pouvez :

  • Modifier les termes du glossaire.
  • Supprimer les termes du glossaire.
  • Exporter le glossaire d'un site pour l'importer dans un autre site.
  • Changer les permissions de l'outil Glossaire pour tous les utilisateurs dans un rôle donné.
  • L'outil Glossaire se trouve généralement dans la catégorie Guide sur le site web de portfolio.

Pour accéder à l'outil de glossaire, cliquez sur le bouton Démarrer le glossaire. Sakai affiche la page d'accueil Glossaire.

La page d'accueil contient une liste de termes du glossaire par ordre alphabétique, avec leurs définitions. Vous avez plusieurs options pour visionner le glossaire :

  • Pour voir la définition longue ou la description courte d'un terme, cliquez sur le terme lui-même. Sakai affiche la définition complémentaire via un hyperlien vers une fenêtre séparée. Pour fermer la fenêtre, cliquez sur Fermer
  • Pour afficher la première page de termes, cliquez sur Premier
  • Pour afficher la page précédente de termes, cliquez sur Précédent
  • Pour afficher la page suivante de termes, cliquez sur Suivant
  • Pour afficher la dernière page de termes, cliquez sur dernier
