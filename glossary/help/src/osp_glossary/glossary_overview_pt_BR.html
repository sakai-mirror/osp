Glossário: Resumo

A ferramenta Glossário oferece definições para linha e nomes de colunas em matrizes. Ao fazer isso, ele compensa o espaço limitado permitido para linha e nomes de coluna.

Quando os participantes movem o mouse sobre uma linha ou nome da coluna em uma matriz, o Sakai exibe uma breve definição ou explicação que foi inserida por meio da ferramenta Glossário. Quando os participantes clicam sobre o nome, hiperlinks para informações mais detalhadas escolhidas pelo organizador do site. Os organizadores do site costumam usar esses hiperlinks para fornecer informações detalhadas sobre os objetivos, resultados, ou padrões representados por cada critério, uma lista completa das atividades do campus para cada categoria, ou um agrupamento extensivo de recursos para desenvolver cada habilidade.

Se você é um administrador de sistema ou um instrutor, a ferramenta Glossário permite que você crie definições curta e longa, e explicações descritas acima. Se você é um administrador de sistema, você também pode fazer termos do glossário global. Neste caso, os termos serão exibidos em todos os sites que usam a ferramenta Glossário no Sakai.

Esta ferramenta também permite que você gerencie termos do glossário de outras formas. Você pode:

  • Alterar termos do glossário.
  • Deletar termos do glossário.
  • Use um glossário criado em outro site para exportar de um site e importar no outro.
  • Altere as permissões da ferramenta Glossário para todos os usuários conforme o papel.
  • A ferramenta Glossário geralmente está localizada na guia categoria no site do portfólio.

Para acessar a ferramenta Glossário, clique no botão Iniciar para Glossário. O Sakai exibirá a página inicial do Glossário.

A página inicial contém uma lista de termos do glossário em ordem alfabética, juntamente com suas breve definições. Você tem várias opções para visualizar o glossário:

  • Para ver a definição longa ou descrição de um termo, clique no próprio termo. O Sakai exibirá a definição com hiperlink em uma janela separada. Para fechar a janela, clique em Fechar.
  • Para exibir a primeira página de termos, clique em Primeiro.
  • Para exibir a página anterior de termos, clique em Anterior.
  • Para exibir a próxima página de termos, clique em Avançar.
  • Para exibir a última página de termos, clique em Última.
