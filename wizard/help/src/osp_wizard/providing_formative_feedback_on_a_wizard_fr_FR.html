Apporter un retour d'information formateur avec l'assistant de saisie

Si vous avez les permissions appropriées pour être relecteur, vous pouvez passer en revue le travail de tout participant avec tout assistant de saisie du site et lui apporter un retour d'information. Le travail de chaque participant est affiché en lecture seule.

Pour apporter un retour d'information, vous remplissez un formulaire qui est joint à l'assistant dans son ensemble ou à une page particulière et donnez à votre formulaire un titre unique une fois achevé. Ce formulaire terminé est alors disponible pour le participant du site. Vous pouvez remplir le même formulaire autant de fois que vous voulez (avec un titre différent à chaque fois) pour apporter un retour d'information sur les changements effectués par le participant. Vous pouvez passer en revue et apporter un retour d'information quand vous le souhaitez jusqu'à ce que le participant ait soumis l'assistant ou la page de l'assistant pour une évaluation.

Pour passer en revue et effectuer un retour d'information, accédez à l'outil des assistants et suivez ces étapes :

  1. Si la page d'accueil des assistants n'est pas affichée, cliquez sur le bouton "Réinitialiser".
  2. Utilisez la liste déroulante pour sélectionner le nom de la personne de qui vous souhaitez relire.
  3. Cliquez sur "Aller à". Sakai affiche une liste des assistants de l'utilisateur choisi.
  4. Trouvez l'assistant que vous souhaitez passer en revue et cliquez sur son nom. Sakai affiche la page d'introduction de l'assistant.
  5. Observez la page d'introduction et ensuite cliquez sur "Suivant". Sakai affiche la première page de l'assistant. Si ce n'est pas la page sur laquelle vous souhaitez travailler, cliquez sur "Suivant" jusqu'à atteindre la page appropriée.
  6. Pour évaluer un item joint, cliquez sur son nom.
  7. Pour insérer un retour d'information, cliquez sur le lien "Ajouter un retour d'information" dans la section "Retour d'information" en bas de la page. Sakai affiche un écran contenant un formulaire de retour d'information.
  8. Entrez les informations dans les champs de saisie appropriés. Attention ! La section de l'écran "Droit d'accès" vous permet de rendre public votre retour d'information (disponible pour d'autres participants que celui pour lequel vous vérifiez le travail). En règle générale, nous vous recommandons de ne pas rendre public le retour d'information. Pour que votre relecture reste entre vous et le participant, ne cliquez pas non plus sur le bouton "radio" dans cette section.
  9. Pour ajouter votre retour d'information, cliquez sur "Ajouter". Sakai affiche la page des assistants à nouveau. Le titre de votre retour d'information est affiché dans la partie de l'écran destinée aux retours d'informations.
  10. Choisissez parmi ces options:
    • Pour voir d'autres pages de l'assistant de ce participant, cliquez sur "Suivant".
    • Pour revenir à la page d'accueil des assistants, cliquez sur "Revenir aux assistants".
    • Si vous avez atteint la dernière page de l'assistant, cliquez sur "Terminer" pour revenir à la page d'accueil des assistants.
