Alterar Permissões

Para alterar as permissões padrão da ferramenta Assistente, acesse a ferramenta e siga os passos abaixo:

  1. Se a barra de menu de Assistentes não for exibida, clique no botão Reset.
  2. Clique no botão Permissões no menu bar. O Sakai exibirá a tela Permissões.
  3. Referindo-se à tabela abaixo, as caixas de seleção para selecionar as permissões adequadas para os usuários em cada papel.



    Permissão Descrição
    Publicar Os usuários neste papel podem publicar assistentes para uso dos participantes do site.
    Deletar Os usuários nesse papel podem excluir assistentes não publicados.
    Criar Os usuários nesse papel podem criar e importar assistentes para uso próprio e/ou dos participantes do site.
    Editar Os usuários nesse papel podem mudar qualquer componente dos assistentes não publicados e um número limitado de componentes de assistentes publicados.
    Revisar Os usuários nesse papel podem rever o trabalho do participante em andamento nos assistentes e oferecer feedback formativo sobre ele.
    Avaliar Os usuários nesse papel podem rever o trabalho que os participantes submeteram através das páginas do assistente ou assistentes como um todo e oferecer feedback somativo sobre ele.
    Visualizar Os usuários nesse papel podem visualizar e completar os assistentes publicados que tenham sido criados para seu uso.
    Exportar Os usuários nesse papel podem exportar um assistente para suas áreas de trabalho como um arquivo compactado.


  4. Para salvar as configurações de permissões, clique em salvar. O Sakai salva as configurações e exibe a página inicial de Assistentes. Ou, para retornar para a tela sem salvar as alterações, clique em cancelar.
