Exportar e Importar um Assistente

Se você possuir mais de um site, você pode usar o mesmo assistente em dois ou mais deles. Para isso, primeiro você exportar o assistente para sua área de trabalho como um arquivo compactado e depois importa para outro site(s) onde você pretende utilizá-lo.

Exportar um Assistente

Para exportar um assistente como um arquivo compactado, acesse a ferramenta Assistentes e siga estes passos:

  1. Se a página inicial de Assistentes não for exibida, clique no botão Redefinir.
  2. Encontre o assistente que você deseja exportar.
  3. Clique no link Exportar abaixo o nome do assistente. O Sakai exibe a janela Abrir Arquivo.
  4. Siga as instruções.
  5. Faça o upload do arquivo compactado para Repositório.

Importar um Assistente

Para importar um assistente da área de trabalho como um arquivo compactado, acesse o site ao qual você deseja importar e siga estes passos:

  1. Adicione o arquivo em Repositório.
  2. Acesse a ferramenta Assistentes. Se a barra de menu de Assistentes não for exibida, clique no botão Reset.
  3. Clique no botão Importar na barra de menu. O Sakai exibirá a tela Importar Assistente.
  4. Clique em Selecionar arquivos. O Sakai exibirá a tela Selecionar Item.
  5. Selecione o arquivo compactado e siga as instruções para selecionar um item. Depois clique em Concluir, o Sakai exibirá a tela Importar Assistente novamente.
  6. O nome do arquivo que você escolheu é exibido na caixa Nome.
  7. Clique no botão Importar. Sakai importa o arquivo da área do Repositório e exibe novamente a página inicial do Assistentes. O assistente de importação estará listado. Ele tem o mesmo nome e instruções como tinha em seu site original. Ou, para voltar à página inicial sem importar o arquivo para a ferramenta Formulários, clique em Cancelar.

Independentemente do estado (publicado ou não) do assistente no site original, é sempre importado no estado não publicado.
