Assistentes: Resumo

Assistentes são coleções personalizadas de telas informativas e formulários que fornecem estrutura e orientação para uma ampla variedade de atividades e processos.

Se você é um administrador de sistema ou um organizador de site, você pode usar essa ferramenta para realizar essas atividades.

  • Criar, alterar e deletar assistentes.
  • Publicar um assistente para que os participantes do site usem.
  • Usar um assistente criado em um site em outro site, exportando do primeiro site e importando no outro site.
  • Alterar as permissões da ferramenta Assistente para todos os usuários em qualquer papel atribuido.

Se você é um participante do site, você usa a ferramenta para completar assistentes.

Tipos de Assistentes

No Sakai, você pode criar três tipos de assistentes. O tipo mais simples é um assistente sequencial. Este tipo de assistente é bem adequado para atividades em que uma série de passos são realizados de uma forma linear. Por exemplo, você pode criar um assistente sequencial para orientar os alunos através das etapas de criação de um papel termo ou para orientar os alunos que uma experiência que se estende por um período significativo de tempo. Um assistente de um experimento pode controlar o tempo de cada passo, coletar dados e interpretações dos alunos em pontos críticos, e oferecer uma oportunidade para você ou outras pessoas para fornecer feedback ao longo do caminho.

O segundo tipo de assistente é hierárquico. Um assistente hierárquico apresenta formulários e informações como um conjunto de categorias e subcategorias, bem como um esboço que traz informações. Assistentes hierárquicos são úteis na organização e acompanhamento atividades, cada qual tem etapas originais. Você pode criar um assistente hierárquico para acompanhar o pessoal, atividades acadêmicas, profissionais, e extracurricular para uso dos assessores acadêmicos ou para uso em currículos em desenvolvimento.

O terceiro tipo de assistente é um assistente da matriz. Também chamado de matriz, este assistente apresenta formulários e informações em um conjunto de linhas e colunas. Você pode criar uma matriz para acompanhar o progresso em diversas disciplinas (as linhas), que têm etapas semelhantes (as colunas). Cada célula da matriz é um ponto no qual você pode acompanhar resultados, incentivar a reflexão sobre como as disciplinas e fornecer feedback.

Quem Usa Assistentes

Na versão out-of-the-box do Sakai, qualquer participante do site pode criar um assistente em Meu Site. Mais frequentemente, entretanto, instrutores e os organizadores do site criam e compartilham assistentes com os participantes do site no portfolio, curso e projeto de site. Depois que um assistente foi publicado, os usuários de vários papéis podem acessá-lo.

Se você é um orientador acadêmico e criou um assistente em um site portfolio, pode acompanhar o progresso do aluno em um programa específico. Cada aluno no programa acessa o seu assistente para preencher formulários, anexar arquivos, como prova de trabalho concluído, refletir sobre a aprendizagem, etc. Os revisores e avaliadores (que você identifica quando você cria o assistente) também acessa o assistente para rever o trabalho e adicionar comentários em pontos-chave após o assistente inteiro estiver concluído.

Componentes dos Assistentes

Um assistente é constituído por uma ou mais unidades conhecidas como páginas. Cada página pode conter muitos componentes diferentes. Estes componentes podem incluir instruções ou orientação geral para o usuário, uma introdução ou justificativa para a página, exemplos para ajudar os participantes a entender como usar o assistente, formulários para as pessoas em vários papéis completar (incluindo, se desejado, formulários que solicitam reflexão ou outro comentário), e/ou itens de Repositório.

Para construir um assistente eficaz, você usar a programação XSD e a ferramenta Formulários para criar formulários. Os formulários criados permitem a você estruturar e coletar informações, tais como reflexões sobre a aprendizagem e feedback e avaliação do trabalho exibido no assistente. Estes formulários podem ser aplicados para o assistente como um todo e/ou para cada página do assistente. Você também pode utilizar um arquivo .css para criar um estilo de visualização para o assistente.

A ferramenta Assistentes lhe permite criar, modificar, publicar, importar, exportar e assistentes. Esta ferramenta é normalmente localizado na categoria Recolher & Refletir em um site de portfólio, na categoria Guia & Instrução em um site de curso, e na categoria Colaboração em um site de projeto.

Para acessar a ferramenta, clique no botão Iniciar para Assistentes. O Sakai exibirá a página inicial dos Assistentes.

A página inicial também é conhecida como a tela do Gerenciador de Assistente. Ela contém uma lista dos assistentes que você criou. Para cada assistente, você pode ver o seu nome, uma breve descrição de sua finalidade, seu status (publicado ou não publicado), seu tipo (sequencial ou hierárquico), o número de páginas que foram apresentadas por todos os participantes para a avaliação, o número total de páginas do assistente, e a data em que foi acessado pela última vez.
