Exportation et importation d'un assistant de saisie

Si vous possédez plus d'un site, vous pourriez vouloir utiliser le même assistant dans l'un ou l'autre de ces sites. Pour ce faire, vous devez en premier lieu exporter l'assistant sur votre bureau en tant que fichier compressé et ensuite l'importer dans l'autre site sur lequel vous prévoyez de l'utiliser.

Exporter un assistant

Pour exporter un assistant en tant que fichier compressé, accédez à l'outil des assistants et suivez ces étapes :

  1. Si la page d'accueil des assistants n'est pas affichée, cliquez sur le bouton "Réinitialiser".
  2. Trouvez l'assistant que vous voulez exporter.
  3. Cliquez sur le lien "Exporter" en dessous de son nom. Sakai affiche une fenêtre "Ouverture d'un fichier".
  4. Suivez les instructions.
  5. Téléversez le fichier compressé aux Ressources.

Importer un assistant

Pour importer un assistant à partir de votre bureau en tant que fichier compressé, accédez au site sur lequel vous souhaitez l'importer et suivez ces étapes :

  1. Ajoutez le fichier aux Ressources.
  2. Accédez à l'outil des assistants. Si la barre de menu des assistants n'est pas affichée, cliquez sur le bouton "Réinitialiser".
  3. Cliquez sur le bouton "Importer" de la barre de menu. Sakai affiche l'écran d'importation d'assistant.
  4. Cliquez sur "Sélectionner un fichier". Sakai affiche l'écran de sélection d'item.
  5. Sélectionnez le fichier compressé pour l'assistant en suivant les instructions données pour sélectionner un item. Après que vous ayez cliqué sur "Terminer", Sakai affiche de nouveau l'écran d'importation d'assistant.
  6. Le nom du fichier que vous avez choisi est affiché dans la case du nom.
  7. Cliquez sur le bouton "Importer". Sakai importe le fichier à partir des ressources et affiche de nouveau la page d'accueil des assistants. L'assistant importé y est listé. Il détient le même nom et les mêmes instructions que sur son site d'origine. Sinon, pour revenir à la page d'accueil sans importer le fichier dans l'outil, cliquez sur "Annuler".

Quelque soit l'état de l'assistant sur son site d'origine (publié ou pas), il est toujours importé dans l'état non publié.
