Deletar um Assistente

Você pode excluir um assistente só se ele ainda não tiver sido publicado. Para excluir um assistente, acesse a página inicial de Assistentes e encontre o assistente que deseja excluir. Em seguida, siga estes passos:

  1. Clique no link Excluir abaixo do nome do assistente. O Sakai exibe esta tela:
  2. Para excluir este assistente, clique em Excluir. O Sakai exclui o assistente e exibe a página inicial de Assistentes. Ou, para retornar para a página inicial sem excluir este assistente, clique em Cancelar.
