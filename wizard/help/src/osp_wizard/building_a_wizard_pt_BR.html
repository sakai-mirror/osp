Construir uma Assistente

Para construir uma matriz de assistente, use a ferramenta Matrizes.

Para construir um assistente sequencial ou hierárquico, acesse a ferramenta Assistentes e siga estes passos:

Passo 1: Criar o Assistente

  1. Se a barra de menu de Assistentes não for exibida, clique no botão Redefinir.
  2. Clique no botão Adicionar no menu bar. O Sakai exibirá a tela Adicionar.
  3. Selecione o tipo de assistente que você deseja criar clicando no botão para sequencial ou hierárquico.
  4. Para prosseguir, clique em Continuar. Ou, para voltar à página inicial de Assistentes sem criar um assistente, clique em Cancelar.
    5. Se você clicou em Continuar, o Sakai exibirá a tela para Adicionar Assistente, passo 1.
  5. Usando as caixas mostradas, digite um nome, uma descrição do objetivo do assistente, e palavras-chave. Apenas o nome é obrigatório.
  6. Escolha uma das seguintes opções:
    • Para aplicar o estilo padrão do Sakai para este assistente, pule esta etapa e vá direto para o passo 7.
    • Para aplicar outro estilo para este assistente, clique em Selecionar Estilo. O Sakai exibe a tela Gerenciar Estilos. Esta tela lista os estilos disponíveis para uso de seus assistentes. Clique no link Selecionar para o estilo que deseja aplicar a este assistente. O Sakai reexibe a tela para Adicionar Assistentes, passo 1. O nome do estilo que você escolheu é exibido na caixa Aplicar um Estilo. Ou, para voltar a tela sem selecionar um estilo, clique em Voltar.
  7. Para prosseguir, clique em Continuar. Ou, para voltar à página inicial de Assistentes sem criar um assistente, clique em Cancelar.
    8. Se você clicou em Continuar, o Sakai exibe a tela para Adicionar Assistente, passo 2. Esta tela é apresentada na próxima subseção.

Passo 2: Adicionar Suporte de Assistente

Neste ponto, você precisa adicionar informações e formulários que se aplicam ao assistente como um todo. Para fazer isso, siga estes passos:

  1. Para adicionar instruções e outras informações úteis para os usuários, clique em Adicionar Instruções. Sakai exibe a tela de Orientação.
  2. Escolha uma ou ambas as opções:
    • Digite o texto diretamente na caixa de texto na tela. Use os ícones e as listas suspensa para formatar o texto conforme o caso.
    • Anexar um arquivo, clicando em selecionar itens. Para fazer isso, siga as instruções dadas para selecionar um item. Depois que você clicar em Concluir, Sakai exibe a tela de Orientação novamente.
  3. Para salvar as informações que você digitou ou anexou, clique em Salvar. Sakai salva as informações no assistente e exibe a tela Adicionar Assistente novamente, passo 2. Se você tiver inserido o texto, o texto é exibido na parte apropriada da tela. Se você tiver anexado um arquivo, o nome é apresentado.
  4. Para explicar o objetivo do assistente para os participantes do site, clique em Adicionar Justificativa. O Sakai exibe a tela Adicionar Justificativa.
  5. Para adicionar exemplos para mostrar aos usuários como usar o assistente, clique em Adicionar Exemplos. Sakai exibe a tela de Adicionar Exemplo.
  6. Para adicionar um formulário para recolher reflexões dos participantes sobre o trabalho do assistente, selecione na lista de opções de Reflexão.
  7. Para adicionar um formulário no qual os revisores fornecerão feedback formativo para os participantes, selecione o formulário apropriado na lista suspensa Feedback.
  8. Para adicionar um formulário no qual os revisores fornecem feedback somativo aos participantes, selecione o formulário de avaliação apropriado na lista suspensa. Se você quiser que os participantes sejam capazes de modificar suas respostas, não inclua um formulário de avaliação e identifique avaliadores para o assistente. Na ausência de formulário de avaliação e um avaliador para fornecer feedback somativo, o assistente não tem pontuação final, de modo que os participantes podem voltar e alterá-lo conforme desejado.
  9. Para selecionar as pessoas para fornecer feedback somativo, clique em Selecionar Avaliadores. Sakai exibe a tela de Selecionar Avaliadores. A janela do lado esquerdo da tela, lista todas os papéis definidas para este site. Abaixo da linha, ele também lista todos os participantes do site.
  10. Para selecionar os avaliadores, clique nos nomes dos papéis e/ou indivíduos e, em seguida, clique no botão Adicionar. Os papéis que você selecionou para os avaliadores deve ter permissão para avaliar assistentes. Além disso, os indivíduos selecionados para a avaliação deve ter o papel atribuído selecionado para avaliação no site. Se essas condições não forem atendidas, os usuários selecionados não serão capazes de ver e avaliar os assistentes. Para mais informações sobre como definir permissões, veja "Alterar permissões para a ferramenta Assistentes". Para selecionar mais de um papel ou pessoa de cada vez, mantenha a tecla CTRL pressionada enquanto clica. Sakai exibe os nomes dos papéis e/ou pessoas na caixa à direita.
  11. Para remover qualquer um desses papéis ou pessoas da lista de avaliadores selecionados, clique em seus nomes e clique no botão Remover.
  12. Quando você tiver identificado todos os papéis e/ou participantes individuais que irão atuar como avaliadores, clique em Concluir. O Sakai exibe a tela Adicionar Assistente, etapa 2, novamente. A tela mostra os nomes dos papéis e/ou indivíduos selecionados como avaliadores.
  13. Escolha uma das seguintes opções:
    • Para criar páginas adicionais para este assistente, clique em Continuar. O Sakai exibe a tela Adicionar Assistente, etapa 3. Esta tela varia, dependendo se você está criando um assistente sequencial ou hierárquico.
    • Para completar este assistente sem adicionar páginas adicionais, clique em Concluir. O Sakai exibe a página inicial do Assistentes. Seu novo assistente está listado na tela.
    • Para retornar a tela Adicionar Assistente, etapa 1, clique em Voltar.
    • Para sair sem criar este assistente, clique em Cancelar. O Sakai exibe a página inicial do Assistentes.
  14. Para adicionar páginas para este assistente, siga as instruções no "Passo 3: Adicionando páginas a um Assistente sequencial" ou "Passo 4: Adicionando categorias, subcategorias e páginas de um Assistente hierárquico".
  15. Para publicar esse assistente, consulte "Publicar um Assistente"

Passo 3: Adicionar Páginas ao Assistente Sequencial

Se você estiver criando um assistente sequencial e concluiu o "Passo 2: Adicionar Suporte de Assistente", ao clicar em Concluir, o Sakai exibe esta tela:

  1. Clique no botão Adicionar na barra de menu. Sakai exibe a tela Rever Assistente de Página de Configurações. Esta tela se parece com uma combinação da Adicionar Assistente do passo 1, e Adicionar Assistente do passo 2. Assim como as telas forneceram todas as caixas, listas suspensa e acesso a arquivos e formulários necessários para que o assistente como um todo, esta tela fornece as caixas que você precisa para definir uma única página no assistente. Esta tela contém uma caixa nova, que é uma lista de opções para adicionar formulários para os participantes do site para usarem para completar esta página do assistente.
  2. Referindo-se às instruções dadas no "Passo 1: Criar o Assistente" e "Passo 2: Adicionar Suporte de Assistente", forneça as informações e materiais que os participantes, revisores e avaliadores assistente precisam para esta página.
  3. Para selecionar formulários para os participantes completarem, selecione um formulário a partir da lista suspensa Formulários e clique em Adicionar. Sakai exibe o nome do formulário que você adicionou. Repita este passo para selecionar outros formulários que você precisa.
  4. Após digitar as informações, anexar formulários e arquivos, e identificar os avaliadores, conforme necessário, clique em Salvar alterações. O Sakai exibe a tela Adicionar Assistente, passo 3. A página que você adicionou está listada na tela. Ou, para retornar a tela sem adicionar a página para o assistente, clique em Cancelar.
  5. Para adicionar outra página, repita os passos 1 a 4.
  6. Depois de adicionar as páginas como você precisa, clique em Concluir. O Sakai exibe a página inicial de Assistentes. Ou, para retornar a tela Adicionar Assistente, passo 2, clique em Voltar. Para retornar à página inicial de Assistentes sem adicionar esse assistente, clique em Cancelar.

Passo 4: Adicionar Categorias, Subcategorias, e Páginas ao Assistente Hierarquico

Se você estiver criando um assistente hierárquico e concluiu o "Passo 2: Adicionar Suporte de Assistente", ao clicar em Concluir, o Sakai exibe a tela Adicionar Assistente Hierárquico, Passo 3. Essa tela fornece o ponto de partida a partir do qual você adicionar categorias, subcategorias e páginas para o seu assistente hierárquico.

Siga as instruções nas subseções abaixo. Após terminar seu trabalho, escolha uma destas opções:

  • Para salvar seu trabalho e retornar à página inicial do Assistentes, clique em Concluir.
  • Para salvar seu trabalho e retornar a tela Adicionar Assistente, Passo 2, clique em Voltar.
  • Para retornar à página inicial sem salvar nenhuma das categorias, subcategorias e páginas que você criou, clique em Cancelar.

Adicionar Categorias

Para adicionar uma ou mais categorias, acesse a tela Adicionar Assistente Hierárquico, passi 3, e siga estes passos:

  1. Clique no link Adicionar Categoria para o assistente. Sakai exibe a tela para Adicionar Categoria ao Assistente
  2. Digite um título para a categoria na caixa Título. Esta entrada é obrigatória.
  3. Se desejar, digite uma descrição da categoria na caixa Descrição e insera palavras-chave na caixa Palavras-chave.
  4. Para salvar essa categoria, clique em Salvar. Ou, para retornar a tela Adicionar Assistente, passo 3, sem incluir aa categoria, clique em Cancelar. Se você clicou em Salvar, o Sakai salva a categoria e exibe a tela Adicionar Assistente, passo 3, novamente. A categoria que você adicionou está listada.
  5. Para adicionar outra categoria, repita os passos 1 a 4.

Adicionar Subcategorias

Para adicionar uma ou mais subcategorias, acesse a tela Adicionar Assistente Hierárquico, passo 3, e siga estes passos:

  1. Encontre a categoria para a qual você deseja adicionar uma subcategoria e clique no link Adicionar Categoria. O Sakai exibe a tela Adicionar Categoria ao Assistente
  2. Digite um título para a subcategoria na caixa Título. Esta entrada é obrigatória.
  3. Se desejar, digite uma descrição da subcategoria na caixa Descrição e insera palavras-chave na caixa de Palavras-Chave.
  4. Para salvar essa subcategoria, clique em Salvar. Ou, para retornar a tela Adicionar Assistente, passo 3, sem adicionar a subcategoria, clique em Cancelar. Se você clicou em Salvar, o Sakai salva a subcategoria e exibe a tela Adicionar Assistente, passo 3, novamente. A subcategoria que você adicionou está listada.
  5. Para adicionar outra subcategoria, repita os passos 1 a 4.

Adicionar Páginas

Para adicionar uma página para o assistente, acesse a tela Adicionar Assistente Hierárquico, Etapa 3, e siga estes passos:

  1. Dependendo do local onde deseja adicionar uma página, clique no link Adicionar Página para o próprio assistente ou de qualquer categoria ou subcategoria no assistente. Sakai exibe os Rever Configurações da Página do Assistente
  2. Referindo-se às instruções dadas no "Passo 1: Criar o Assistente" e "Passo 2: Adicionar Suporte de Assistente", forneça as informações e materiais que os participantes, revisores e avaliadores do assistente precisam para esta página.
  3. Para selecionar formulários para os participantes completarem, selecione um formulário a partir da lista suspensa Formulários e clique em Adicionar. Sakai exibe o nome do formulário que você adicionou. Repita esse passo para selecionar os outros formulários que você precisa.
  4. Após digitar as informações, anexar formulários e arquivos, e identificar os avaliadores, conforme necessário, clique em Salvar Alterações. O Sakai exibe a tela Adicionar Assistente, etapa 3. A página que você adicionou está listada na tela. Ou, para retornar atela sem adicionar a página para o assistente, clique em Cancelar.
  5. Depois de adicionar as páginas como você precisa, clique em Concluir. O Sakai exibe a página inicial de Assistentes. Ou, para retornar a tela Adicionar Assistente, passo 2, clique em Voltar. Para retornar à página inicial de Assistentes sem adicionar esse assistente, clique em Cancelar.
  6. Após digitar as informações, anexar formulários e arquivos, e identificar os avaliadores, conforme necessário, clique em Salvar Alterações. O Sakai exibe a tela Adicionar Assistente, etapa 3. A página que você adicionou está listada na tela. Ou, para retornar atela sem adicionar a página para o assistente, clique em Cancelar.
  7. Para adicionar outra página ao assistente, repita os passos 1 a 6.
