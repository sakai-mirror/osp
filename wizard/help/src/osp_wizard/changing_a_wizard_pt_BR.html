Alterar um Assistente

A ferramenta Assistentes permite que você altere assistentes por estas maneiras:

  • Alterar o nome ou outros atributos do assistente.
  • Alterar o conteúdo das páginas ou categorias do assistente
  • Alterar a ordem das páginas nas categorias (apenas assistentes hieráquico)
  • Alterar a ordem das categorias no assistente
  • Mover páginas para outras categorias
  • Deletar páginas e categorias individuais

Para fazer qualquer uma dessas alterações, acesse a ferramenta Assistentes e siga estes passos:

  1. Se a página inicial de Assistentes não for exibida, clique no botão Redefinir.
  2. Encontre o assistente que deseja alterar e clique no link Rever abaixo do seu nome. O Sakai exibe a tela Rever Assistente, Passo 1. Exceto para o seu nome, esta tela se parece com a tela Adicionar Assistente, Passo 1.
  3. Referindo-se às instruções dadas no "Passo 1: Criar o Assistente", passos 5 a 7, altere as entradas conforme apropriado.
  4. Se você pressionar Continuar, o Sakai exibe a tela Rever Assistente, passo 2. Exceto para o seu nome, esta tela é idêntica à tela Adicionar Assistente, passo 2.
  5. Referindo-se as instruções em "Passo 2: Adicionar Suporte ao Assistente", altere as entradas conforme o caso. Se você pressionar Continuar, o Sakai exibe a tela Rever Assistente, passo 3. Esta tela varia, dependendo se você está revisando um assistente sequencial ou hierárquico. Observe que as categorias e páginas são identificadas por ícones diferentes:
    • triângulo com face para baixo denota uma categoria que contém páginas.
    • triângulo com face para direita enota uma categoria que não contém páginas.
    • círculo denota uma página.
  6. Para fazer outros tipos de alterações, escolha uma ou mais destas opções:
    • Para adicionar páginas a um assistente sequencial, consulte "Passo 3: Adicionar Páginas a um Assistente Sequencial".
    • Para adicionar categorias e/ou páginas de um assistente hierárquico, consulte "Passo 4: Adicionar Categorias, Subcategorias, e páginas para um Assistente Hierárquico".
    • Para alterar, excluir ou mover uma categoria ou página, veja as subseções a seguir.
  7. Saia da tela em uma das seguintes maneiras:
    • Para salvar seu trabalho e retornar à página inicial do Assistentes clique em Concluir.
    • Para salvar seu trabalho e retornar a tela Adicionar Assistente, Passo 2, clique em Voltar.
    • Para retornar à página inicial sem salvar nenhuma das alterações, clique em Cancelar.

Alterar uma Categoria

Para alterar uma categoria em um assistente hierárquico, siga estes passos:

  1. Na tela Adicionar Assistente, passo 3, encontre a categoria que pretende alterar e clique no link Rever abaixo do seu nome. O Sakai exibe a tela Rever Categoria do Assistente. Exceto por seu nome, é idêntica à tela para Adicionar Categoria ao Assistente.
  2. Referindo-se as instruções obtidas em "Adicionar Categorias," passos 2 a 5, faça as alterações necessárias.

Alterar uma Página

Para alterar uma página no assistente, siga estes passos:

  1. Na tela Adicionar Assistente, passo 3, encontre a página que pretende alterar e clique no link Rever abaixo do seu nome. O Sakai exibirá a tela Rever Página de Configurações do Assistente (veja página 1).
  2. Referindo-se às instruções dadas no início do "Passo 1: Criar o Assistente" com o passo 5, e "Passo 2: Adicionando Suporte ao Assistente", fazer as alterações necessárias.
  3. Para selecionar formas para os participantes completarem, selecione um formulário a partir da lista suspensa Formulários e clique em Adicionar. O Sakai exibe o nome do formulário que você adicionou. Repita este passo para selecionar outros formulários que você precisa.
  4. Após digitar as informações, anexar formulários e arquivos, e identificar os avaliadores, conforme necessário, clique em Salvar Alterações. O Sakai exibe a tela Adicionar Assistente, passo 3. A página que você adicionou estará listada na tela. Ou, para retornar a esta tela sem adicionar a página para o assistente, clique em Cancelar.

Deletar uma Categoria ou Página

Você pode excluir apenas categorias ou páginas não publicadas ou assistentes de pré-visualização. Para excluir uma categoria ou uma página, siga estes passos:

  1. Na tela Adicionar Assistente, passo 3, encontre a categoria ou página que pretende deletar. Atenção: Certifique-se de ter escolhido a categoria ou página correta.
  2. Clique no link Excluir abaixo do nome dessa categoria ou página. O Sakai exclui sua seleção imediatamente, sem pedir-lhe para confirmar sua seleção.

Altear a Ordem das Categorias or Páginas

Para alterar a ordem das páginas em uma categoria ou um assistente sequencial, ou para mudar a ordem das categorias, clique nas setas para cima e/ou para baixo. Cada vez que você clicar em uma seta, o Sakai move a categoria associada ou página para cima ou para baixo na lista.

Mover uma Página ou Categoria para um Outro Nível ou Categoria

Em um assistente hierárquico, você pode mover as páginas de uma categoria para outra. Você também pode mover categorias, tornando-as subcategorias ou mover subcategorias tornando-as categorias.

Para mover uma página ou categoria, siga estes passos:

  1. Na tela Adicionar Assistente, passo 3, encontre a categoria ou página que pretende mover e clique no link Rever abaixo do seu nome. O Sakai reexibirá a tela, exibindos os possíveis destinos para as categorias ou páginas.
  2. Encontre a categoria que você deseja mover para dentro de uma categoria ou página e clique no link Mover Categoria Aqui. O Sakai move a categoria ou página para o novo local.
