Modification des permissions

Pour modifier les permissions par défaut de l'outil d'assistant, accédez à l'outil et suivez ces étapes :

  1. Si la barre de menu des assistants n'est pas affichée, cliquez sur le bouton "Réinitialiser".
  2. Cliquez sur le bouton "Permissions" de la barre de menu. Sakai affiche l'écran des permissions.
  3. En vous référant à la table ci-dessous, cochez les cases pour sélectionner les permissions appropriées pour les utilisateurs dans chaque rôle.



    Permission Description
    Publier Les utilisateurs dans ce rôle peuvent publier des assistants pour l'usage des contributeurs du site.
    Effacer Les utilisateurs dans ce rôle peuvent effacer les assistants non publiés.
    Créer Les utilisateurs dans ce rôle peuvent créer et importer des assistants pour leur usage personnel et/ou par les contributeurs du site.
    Editer Les utilisateurs dans ce rôle peuvent modifier n'importe quel composant d'un assistant non publié et certains composants d'assistants déjà publiés.
    Superviser Les utilisateurs dans ce rôle peuvent superviser le travail en cours du contributeur et lui apporter un retour d'information résumé.
    Evaluer Les utilisateurs dans ce rôle peuvent évaluer le travail soumis par les participants via les pages de l'assistant comme un tout et apporter un retour d'information.
    Voir Les utilisateurs dans ce rôle peuvent voir et compléter des assistants publiés qui ont été créés pour leur utilisation.
    Exporter Les utilisateurs dans ce rôle peuvent exporter un assistant sur leur bureau dans un fichier compressé.


  4. Pour sauvegarder les réglages de permissions, cliquez sur "Enregistrer". Sakai enregistre les réglages et affiche la page d'accueil des assistants. Sinon, pour revenir à cet écran sans sauvegarder les modifications, cliquez sur "Annuler".
